# 1528. Shuffle String / Trộn Chuỗi

## Problem Description / Mô tả bài toán
String `s`, array `indices`.
Character `s[i]` moves to `indices[i]`.
Return shuffled string.

---

## Theoretical Foundation / Cơ sơ lý thuyết

### Direct Mapping
Create char array.
`res[indices[i]] = s[i]`.
Convert to string.

### Complexity / Độ phức tạp
- **Time**: O(N).
- **Space**: O(N).

---

## Analysis / Phân tích

### Approach: Array Placement
New array. Loop. Assign.
Mảng mới. Lặp lại. Gán.
