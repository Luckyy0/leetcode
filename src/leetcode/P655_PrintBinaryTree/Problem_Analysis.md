# 655. Print Binary Tree / In Cây Nhị Phân

## Problem Description / Mô tả bài toán
Given the `root` of a binary tree, construct a 2D string array to represent the tree. Return the 2D array.

---

## Theoretical Foundation / Cơ sơ lý thuyết

### DFS with Position Calculation / DFS với Tính Toán Vị Trí
Calculate height, then dimensions. Fill matrix using DFS with column position formula.

### Complexity / Độ phức tạp
- **Time**: O(N * H).
- **Space**: O(N * H).
