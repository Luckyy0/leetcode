# Problem Analysis: Count Elements With Strictly Smaller and Greater Elements

## Problem Description
Count elements `x` such that there exists `y < x` and `z > x` in array.

## Analysis

1.  **Min/Max Check**:
    -   Find `min` and `max` of array.
    -   Any element strictly between `min` and `max` satisfies the condition.
    -   Count such elements.

## Implementation Details
-   Find min, max.
-   Loop count.
