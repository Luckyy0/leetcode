# 1550. Three Consecutive Odds / Ba Số Lẻ Liên tiếp

## Problem Description / Mô tả bài toán
Array `arr`. Return `true` if there are three consecutive odd numbers in the array.

---

## Theoretical Foundation / Cơ sơ lý thuyết

### Iteration
Count consecutive odd numbers while iterating.
If the count reaches 3, return `true`.
If an even number is encountered, reset the count to 0.

### Complexity / Độ phức tạp
- **Time**: O(N).
- **Space**: O(1).

---

## Analysis / Phân tích

### Approach: Linear Scan
Iterate through the array and check each element.
Lặp qua mảng và kiểm tra từng phần tử.
