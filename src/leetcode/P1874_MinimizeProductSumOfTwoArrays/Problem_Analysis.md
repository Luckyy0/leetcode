# Problem Analysis: Minimize Product Sum of Two Arrays

## Problem Description
Rearrange `nums1` to minimize `sum(nums1[i] * nums2[i])`.

## Analysis

1.  **Sorting**:
    -   Sort `nums1` ascending.
    -   Sort `nums2` descending.
    -   Pair smallest `nums1` with largest `nums2`.

## Implementation Details
-   Arrays.sort.
-   Loop.
