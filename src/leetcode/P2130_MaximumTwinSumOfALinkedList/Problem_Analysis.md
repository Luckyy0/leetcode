# Problem Analysis: Maximum Twin Sum of a Linked List

## Problem Description
Linked list even length `n`.
Twin sum: `node[i] + node[n-1-i]`.
Max twin sum.

## Analysis

1.  **Reverse Second Half**:
    -   Find middle.
    -   Reverse second half.
    -   Iterate both halves together. Sum.
