# 389. Find the Difference / Tìm Sự Khác Biệt

## Problem Description / Mô tả bài toán
You are given two strings `s` and `t`.
String `t` is generated by random shuffling string `s` and then add one more letter at a random position.
Bạn được cho hai chuỗi `s` và `t`. Chuỗi `t` được tạo bằng cách xáo trộn ngẫu nhiên chuỗi `s` và sau đó thêm một chữ cái nữa vào một vị trí ngẫu nhiên.

Return the letter that was added to `t`.
Trả về chữ cái đã được thêm vào `t`.

### Example 1:
```text
Input: s = "abcd", t = "abcde"
Output: "e"
Explanation: 'e' is the letter that was added.
```

### Example 2:
```text
Input: s = "", t = "y"
Output: "y"
```

## Constraints / Ràng buộc
- `0 <= s.length <= 1000`
- `t.length == s.length + 1`
- `s` and `t` consist of lowercase English letters.

---

## Theoretical Foundation / Cơ sở lý thuyết

### XOR / Phép XOR
XOR all characters from both strings.
Since `a XOR a = 0`, all paired characters cancel out.
Result is the added character.

### Sum Difference / Hiệu Tổng
Alternative: Sum of ASCII values of `t` - Sum of `s`.
Or frequency count.

### Complexity / Độ phức tạp
- **Time**: O(N).
- **Space**: O(1).

---

## Analysis / Phân tích

### Approach: XOR

**Algorithm**:
1.  `xor = 0`.
2.  Loop `c` in `s`: `xor ^= c`.
3.  Loop `c` in `t`: `xor ^= c`.
4.  Return `(char) xor`.
