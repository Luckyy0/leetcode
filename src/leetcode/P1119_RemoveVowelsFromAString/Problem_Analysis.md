# 1119. Remove Vowels from a String / Xóa Nguyên âm khỏi Chuỗi

## Problem Description / Mô tả bài toán
Given a string `s`, remove the vowels 'a', 'e', 'i', 'o', and 'u' from it, and return the new string.
Cho một chuỗi `s`, hãy xóa các nguyên âm 'a', 'e', 'i', 'o' và 'u' khỏi nó và trả về chuỗi mới.

---

## Theoretical Foundation / Cơ sơ lý thuyết

### String Filtering / Lọc Chuỗi
Iterate chars, append if not vowel.
