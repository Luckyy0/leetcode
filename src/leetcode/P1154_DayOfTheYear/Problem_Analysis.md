# 1154. Day of the Year / Ngày trong Năm

## Problem Description / Mô tả bài toán
Given a string `date` representing a Gregorian calendar date formatted as `YYYY-MM-DD`, return the day number of the year.
Cho một chuỗi `date` đại diện cho một ngày theo lịch Gregorian được định dạng là `YYYY-MM-DD`, hãy trả về số thứ tự ngày trong năm.

---

## Theoretical Foundation / Cơ sơ lý thuyết

### Date Parsing / Phân tích Ngày
Days in month array.
Handle leap year for Feb (29 days).
Sum days of previous months + current day.
