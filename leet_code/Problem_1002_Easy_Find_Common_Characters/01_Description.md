# Result for Find Common Characters
# *Kết quả cho bài toán Tìm Ký tự Chung*

## Description
## *Mô tả*

Given a string array `words`, return *an array of all characters that show up in all strings within the* `words` *(including duplicates)*. You may return the answer in **any order**.
*Cho một mảng chuỗi `words`, trả về *một mảng chứa tất cả các ký tự xuất hiện trong tất cả các chuỗi thuộc* `words` *(bao gồm cả các ký tự chùng lặp)*. Bạn có thể trả về câu trả lời theo **bất kỳ thứ tự nào**.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** words = ["bella","label","roller"]
**Output:** ["e","l","l"]

## Example 2:
## *Ví dụ 2:*

**Input:** words = ["cool","lock","cook"]
**Output:** ["c","o"]

---

## Constraints:
## *Ràng buộc:*

*   `1 <= words.length <= 100`
*   `1 <= words[i].length <= 100`
*   `words[i]` consists of lowercase English letters.
