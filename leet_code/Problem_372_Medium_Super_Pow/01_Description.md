# Result for Super Pow
# *Kết quả cho bài toán Siêu Lũy thừa*

## Description
## *Mô tả*

Your task is to calculate $a^b \pmod{1337}$ where `a` is a positive integer and `b` is an extremely large positive integer given in the form of an array.
*Nhiệm vụ của bạn là tính $a^b \pmod{1337}$ trong đó `a` là một số nguyên dương và `b` là một số nguyên dương cực lớn được cho dưới dạng một mảng.*

## Example 1:
## *Ví dụ 1:*

**Input:** `a = 2, b = [3]`
**Output:** `8`

## Example 2:
## *Ví dụ 2:*

**Input:** `a = 2, b = [1,0]`
**Output:** `1024`

## Example 3:
## *Ví dụ 3:*

**Input:** `a = 1, b = [4,3,3,8,5,2]`
**Output:** `1`

## Constraints:
## *Ràng buộc:*

*   `1 <= a <= 2^31 - 1`
*   `1 <= b.length <= 2000`
*   `0 <= b[i] <= 9`
*   `b` does not contain leading zeros.
