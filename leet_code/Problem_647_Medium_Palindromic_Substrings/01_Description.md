# Result for Palindromic Substrings
# *Kết quả cho bài toán Chuỗi con Đối xứng*

## Description
## *Mô tả*

Given a string `s`, return the number of **palindromic substrings** in it.
*Cho một chuỗi `s`, hãy trả về số lượng **chuỗi con đối xứng** (palindromic substrings) trong đó.*

A string is a **palindrome** when it reads the same backward as forward.
*Một chuỗi là **đối xứng** (palindrome) khi nó đọc xuôi và ngược đều giống nhau.*

A **substring** is a contiguous sequence of characters within the string.
*Một **chuỗi con** (substring) là một dãy các ký tự liên tiếp trong chuỗi.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** s = "abc"
**Output:** 3
**Explanation:** Three palindromic strings: "a", "b", "c".

## Example 2:
## *Ví dụ 2:*

**Input:** s = "aaa"
**Output:** 6
**Explanation:** Six palindromic strings: "a", "a", "a", "aa", "aa", "aaa".

---

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 1000`
*   `s` consists of lowercase English letters.
