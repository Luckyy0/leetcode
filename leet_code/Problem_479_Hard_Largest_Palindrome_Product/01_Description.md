# Result for Largest Palindrome Product
# *Kết quả cho bài toán Tích Palindrome Lớn nhất*

## Description
## *Mô tả*

Given an integer n, return *the **largest palindromic integer** that can be represented as the product of two `n`-digits integers*. Since the answer can be very large, return it **modulo** `1337`.
*Cho số nguyên n, hãy trả về *số nguyên palindrome **lớn nhất** có thể được biểu diễn dưới dạng tích của hai số nguyên `n`-chữ số*. Vì đáp án có thể rất lớn, hãy trả về kết quả **modulo** `1337`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 2`
**Output:** `987`
**Explanation:** 99 x 91 = 9009, 9009 % 1337 = 987

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 1`
**Output:** `9`

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 8`
