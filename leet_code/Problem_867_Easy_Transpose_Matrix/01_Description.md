# Result for Transpose Matrix
# *Kết quả cho bài toán Chuyển vị Ma trận*

## Description
## *Mô tả*

Given a 2D integer array `matrix`, return *the **transpose** of `matrix`*.
*Cho một mảng số nguyên hai chiều `matrix`, hãy trả về *phần **chuyển vị** của `matrix`*.*

The **transpose** of a matrix is the matrix flipped over its main diagonal, switching the matrix's row and column indices.
*Phần **chuyển vị** của một ma trận là ma trận được lật qua đường chéo chính của nó, hoán đổi chỉ số hàng và cột của ma trận.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** matrix = [[1,2,3],[4,5,6],[7,8,9]]
**Output:** [[1,4,7],[2,5,8],[3,6,9]]

## Example 2:
## *Ví dụ 2:*

**Input:** matrix = [[1,2,3],[4,5,6]]
**Output:** [[1,4],[2,5],[3,6]]

---

## Constraints:
## *Ràng buộc:*

*   `m == matrix.length`
*   `n == matrix[i].length`
*   `1 <= m, n <= 1000`
*   `1 <= m * n <= 10^5`
*   `-10^9 <= matrix[i][j] <= 10^9`
