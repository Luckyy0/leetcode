# Result for Convert Sorted List to Binary Search Tree
# *Kết quả cho bài toán Chuyển Đổi Danh Sách Đã Sắp Xếp Thành Cây Tìm Kiếm Nhị Phân*

## Description
## *Mô tả*

Given the `head` of a singly linked list where elements are sorted in **ascending order**, convert it to a **height-balanced** binary search tree.
*Cho `head` của một danh sách liên kết đơn trong đó các phần tử được sắp xếp theo **thứ tự tăng dần**, hãy chuyển đổi nó thành một cây tìm kiếm nhị phân **cân bằng về chiều cao**.*

## Example 1:
## *Ví dụ 1:*

**Input:** `head = [-10,-3,0,5,9]`
**Output:** `[0,-3,9,-10,null,5]`
**Explanation:** One possible answer is `[0,-3,9,-10,null,5]`, which represents the shown height-balanced BST.

## Example 2:
## *Ví dụ 2:*

**Input:** `head = []`
**Output:** `[]`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in `head` is in the range `[0, 2 * 10^4]`.
*   `-10^5 <= Node.val <= 10^5`
