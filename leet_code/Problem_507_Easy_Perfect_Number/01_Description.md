# Result for Perfect Number
# *Kết quả cho bài toán Số Hoàn hảo*

## Description
## *Mô tả*

A **perfect number** is a positive integer that is equal to the sum of its **positive divisors**, excluding the number itself. A divisor of an integer `x` is an integer that can divide `x` evenly.
*Một **số hoàn hảo** là một số nguyên dương bằng tổng các **ước số dương** của nó, ngoại trừ chính nó. Một ước số của số nguyên `x` là một số nguyên chia hết cho `x`.*

Given an integer `n`, return `true` *if* `n` *is a perfect number, otherwise return* `false`.
*Cho một số nguyên `n`, hãy trả về `true` *nếu* `n` *là số hoàn hảo, nếu không trả về* `false`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `num = 28`
**Output:** `true`
**Explanation:** 28 = 1 + 2 + 4 + 7 + 14
1, 2, 4, 7, and 14 are all divisors of 28.

## Example 2:
## *Ví dụ 2:*

**Input:** `num = 7`
**Output:** `false`

## Constraints:
## *Ràng buộc:*

*   `1 <= num <= 10^8`
