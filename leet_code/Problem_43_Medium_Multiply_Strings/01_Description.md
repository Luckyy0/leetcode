# Result for Multiply Strings
# *Kết quả cho bài toán Nhân Chuỗi*

## Description
## *Mô tả*

Given two non-negative integers `num1` and `num2` represented as strings, return the product of `num1` and `num2`, also represented as a string.
*Cho hai số nguyên không âm `num1` và `num2` dưới dạng chuỗi, hãy trả về tích của `num1` và `num2`, cũng dưới dạng chuỗi.*

**Note:** You must not use any built-in BigInteger library or convert the inputs to integer directly.
***Lưu ý:** Bạn không được sử dụng bất kỳ thư viện BigInteger có sẵn nào hoặc chuyển đổi trực tiếp dữ liệu đầu vào thành số nguyên.*

## Example 1:
## *Ví dụ 1:*

**Input:** `num1 = "2"`, `num2 = "3"`
**Output:** `"6"`

## Example 2:
## *Ví dụ 2:*

**Input:** `num1 = "123"`, `num2 = "456"`
**Output:** `"56088"`

## Constraints:
## *Ràng buộc:*

*   `1 <= num1.length, num2.length <= 200`
*   `num1` and `num2` consist of digits only.
*   Both `num1` and `num2` do not contain any leading zero, except the number 0 itself.
