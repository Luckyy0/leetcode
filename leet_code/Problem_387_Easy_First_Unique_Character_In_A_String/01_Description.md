# Result for First Unique Character in a String
# *Kết quả cho bài toán Ký tự Duy nhất Đầu tiên trong Chuỗi*

## Description
## *Mô tả*

Given a string `s`, *find the first non-repeating character in it and return its index*. If it does not exist, return `-1`.
*Cho một chuỗi `s`, hãy *tìm ký tự đầu tiên không lặp lại trong đó và trả về chỉ số của nó*. Nếu nó không tồn tại, trả về `-1`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = "leetcode"`
**Output:** `0`

## Example 2:
## *Ví dụ 2:*

**Input:** `s = "loveleetcode"`
**Output:** `2`

## Example 3:
## *Ví dụ 3:*

**Input:** `s = "aabb"`
**Output:** `-1`

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 10^5`
*   `s` consists of only lowercase English letters.
