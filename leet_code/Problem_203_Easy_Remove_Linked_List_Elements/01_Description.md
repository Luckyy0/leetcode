# Result for Remove Linked List Elements
# *Kết quả cho bài toán Xóa phần tử trong danh sách liên kết*

## Description
## *Mô tả*

Given the `head` of a linked list and an integer `val`, remove all the nodes of the linked list that has `Node.val == val`, and return *the new head*.
*Cho `head` của một danh sách liên kết và một số nguyên `val`, hãy xóa tất cả các nút của danh sách liên kết có `Node.val == val`, và trả về *đầu danh sách mới***.*

## Example 1:
## *Ví dụ 1:*

**Input:** `head = [1,2,6,3,4,5,6], val = 6`
**Output:** `[1,2,3,4,5]`

## Example 2:
## *Ví dụ 2:*

**Input:** `head = [], val = 1`
**Output:** `[]`

## Example 3:
## *Ví dụ 3:*

**Input:** `head = [7,7,7,7], val = 7`
**Output:** `[]`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the list is in the range `[0, 10^4]`.
*   `1 <= Node.val <= 50`
*   `0 <= val <= 50`
