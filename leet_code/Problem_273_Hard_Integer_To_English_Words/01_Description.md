# Result for Integer to English Words
# *Kết quả cho bài toán Đổi Số nguyên sang Tiếng Anh*

## Description
## *Mô tả*

Convert a non-negative integer `num` to its English words representation.
*Chuyển đổi một số nguyên không âm `num` sang biểu diễn bằng từ tiếng Anh.*

## Example 1:
## *Ví dụ 1:*

**Input:** `num = 123`
**Output:** `"One Hundred Twenty Three"`

## Example 2:
## *Ví dụ 2:*

**Input:** `num = 12345`
**Output:** `"Twelve Thousand Three Hundred Forty Five"`

## Example 3:
## *Ví dụ 3:*

**Input:** `num = 1234567`
**Output:** `"One Million Two Hundred Thirty Four Thousand Five Hundred Sixty Seven"`

## Constraints:
## *Ràng buộc:*

*   `0 <= num <= 2^31 - 1`
