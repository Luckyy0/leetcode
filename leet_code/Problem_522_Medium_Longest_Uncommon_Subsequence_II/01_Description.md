# Result for Longest Uncommon Subsequence II
# *Kết quả cho bài toán Dãy con Không chung Dài nhất II*

## Description
## *Mô tả*

Given an array of strings `strs`, return *the length of the **longest uncommon subsequence** between them*. If the longest uncommon subsequence does not exist, return `-1`.
*Cho một mảng các chuỗi `strs`, hãy trả về *độ dài của **dãy con không chung dài nhất** giữa chúng*. Nếu dãy con không chung dài nhất không tồn tại, hãy trả về `-1`.*

An **uncommon subsequence** between an array of strings is a string that is a **subsequence of one string but not the others**.
*Một **dãy con không chung** giữa một mảng các chuỗi là một chuỗi là **dãy con của một chuỗi nhưng không phải của các chuỗi khác**.*

## Example 1:
## *Ví dụ 1:*

**Input:** `strs = ["aba","cdc","eae"]`
**Output:** `3`

## Example 2:
## *Ví dụ 2:*

**Input:** `strs = ["aaa","aaa","aa"]`
**Output:** `-1`

## Constraints:
## *Ràng buộc:*

*   `2 <= strs.length <= 50`
*   `1 <= strs[i].length <= 10`
*   `strs[i]` consists of lowercase English letters.
