# Result for Add Two Numbers II
# *Kết quả cho bài toán Cộng hai số II*

## Description
## *Mô tả*

You are given two **non-empty** linked lists representing two non-negative integers. The most significant digit comes first and each of their nodes contains a single digit. Add the two numbers and return it as a linked list.
*Bạn được cho hai danh sách liên kết **không trống** đại diện cho hai số nguyên không âm. Chữ số có ý nghĩa nhất (MSB) đứng đầu và mỗi nút của chúng chứa một chữ số duy nhất. Hãy cộng hai số đó và trả về kết quả dưới dạng một danh sách liên kết.*

You may assume the two numbers do not contain any leading zero, except the number 0 itself.
*Bạn có thể giả định rằng hai số không chứa bất kỳ số 0 nào ở đầu, ngoại trừ chính số 0.*

## Example 1:
## *Ví dụ 1:*

**Input:** `l1 = [7,2,4,3], l2 = [5,6,4]`
**Output:** `[7,8,0,7]`

## Example 2:
## *Ví dụ 2:*

**Input:** `l1 = [2,4,3], l2 = [5,6,4]`
**Output:** `[8,0,7]`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in each linked list is in the range `[1, 100]`.
*   `0 <= Node.val <= 9`
*   It is guaranteed that the list represents a number that does not have leading zeros.
