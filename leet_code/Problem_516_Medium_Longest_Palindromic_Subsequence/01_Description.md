# Result for Longest Palindromic Subsequence
# *Kết quả cho bài toán Dãy con Palindrome Dài nhất*

## Description
## *Mô tả*

Given a string `s`, find *the longest palindromic **subsequence**'s length in* `s`.
*Cho một chuỗi `s`, hãy tìm *độ dài của **dãy con** palindrome dài nhất trong* `s`.*

A **subsequence** is a sequence that can be derived from another sequence by deleting some or no elements without changing the order of the remaining elements.
*Một **dãy con** là một dãy có thể được tạo ra từ dãy khác bằng cách xóa một số hoặc không xóa phần tử nào mà không thay đổi thứ tự của các phần tử còn lại.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = "bbbab"`
**Output:** `4`
**Explanation:** One possible longest palindromic subsequence is "bbbb".

## Example 2:
## *Ví dụ 2:*

**Input:** `s = "cbbd"`
**Output:** `2`
**Explanation:** One possible longest palindromic subsequence is "bb".

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 1000`
*   `s` consists only of lowercase English letters.
