# Result for Remove Duplicate Letters
# *Kết quả cho bài toán Loại bỏ Các Chữ cái Trùng lặp*

## Description
## *Mô tả*

Given a string `s`, remove duplicate letters so that every letter appears once and only once. You must make sure your result is **the smallest in lexicographical order** among all possible results.
*Cho một chuỗi `s`, hãy loại bỏ các chữ cái trùng lặp sao cho mỗi chữ cái chỉ xuất hiện một lần và duy nhất một lần. Bạn phải đảm bảo kết quả của mình là **nhỏ nhất theo thứ tự từ điển** trong số tất cả các kết quả có thể.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = "bcabc"`
**Output:** `"abc"`

## Example 2:
## *Ví dụ 2:*

**Input:** `s = "cbacdcbc"`
**Output:** `"acdb"`

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 10^4`
*   `s` consists of lowercase English letters.
