# Result for Strobogrammatic Number III
# *Kết quả cho bài toán Số Strobogrammatic III*

## Description
## *Mô tả*

Given two strings `low` and `high` that represent two integers, return *the number of **strobogrammatic numbers** that are in the range* `[low, high]`.
*Cho hai chuỗi `low` và `high` đại diện cho hai số nguyên, hãy trả về *số lượng **các số strobogrammatic** nằm trong phạm vi* `[low, high]`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `low = "50", high = "100"`
**Output:** `3`
**Explanation:** 69, 88, and 96 are three strobogrammatic numbers.

## Example 2:
## *Ví dụ 2:*

**Input:** `low = "0", high = "0"`
**Output:** `1`

## Constraints:
## *Ràng buộc:*

*   `1 <= low.length, high.length <= 15`
*   `low` and `high` consist of only digits.
*   `low <= high`
*   `low` and `high` do not contain any leading zeros except for zero itself.
