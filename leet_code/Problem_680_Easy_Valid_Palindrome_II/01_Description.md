# Result for Valid Palindrome II
# *Kết quả cho bài toán Chuỗi Đối xứng II*

## Description
## *Mô tả*

Given a string `s`, return `true` if the `s` can be palindrome after deleting **at most one** character from it.
*Cho một chuỗi `s`, hãy trả về `true` nếu chuỗi `s` có thể trở thành chuỗi đối xứng sau khi xóa **tối đa một** ký tự từ nó.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** s = "aba"
**Output:** true

## Example 2:
## *Ví dụ 2:*

**Input:** s = "abca"
**Output:** true
**Explanation:** You could delete the character 'c'.

## Example 3:
## *Ví dụ 3:*

**Input:** s = "abc"
**Output:** false

---

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 10^5`
*   `s` consists of lowercase English letters.
