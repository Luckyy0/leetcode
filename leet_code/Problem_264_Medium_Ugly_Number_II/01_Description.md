# Result for Ugly Number II
# *Kết quả cho bài toán Số xấu II*

## Description
## *Mô tả*

An **ugly number** is a positive integer whose prime factors are limited to `2`, `3`, and `5`.
*Một **số xấu** là một số nguyên dương mà các thừa số nguyên tố của nó chỉ giới hạn trong `2`, `3`, và `5`.*

Given an integer `n`, return *the* `n^th` ***ugly number***.
*Cho một số nguyên `n`, hãy trả về ***số xấu*** *thứ* `n`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 10`
**Output:** `12`
**Explanation:** [1, 2, 3, 4, 5, 6, 8, 9, 10, 12] is the sequence of the first 10 ugly numbers.

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 1`
**Output:** `1`
**Explanation:** 1 is typically treated as an ugly number.

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 1690`
