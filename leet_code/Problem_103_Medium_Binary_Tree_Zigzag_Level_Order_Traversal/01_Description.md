# Result for Binary Tree Zigzag Level Order Traversal
# *Kết quả cho bài toán Duyệt Cây Nhị Phân Theo Hình Zigzag*

## Description
## *Mô tả*

Given the `root` of a binary tree, return *the zigzag level order traversal of its nodes' values*. (i.e., from left to right, then right to left for the next level and alternate between).
*Cho `root` của một cây nhị phân, hãy trả về *phép duyệt theo thứ tự tầng hình zigzag của các giá trị nút*. (tức là, từ trái sang phải, rồi từ phải sang trái cho tầng tiếp theo và cứ thế xen kẽ).*

## Example 1:
## *Ví dụ 1:*

**Input:** `root = [3,9,20,null,null,15,7]`
**Output:** `[[3],[20,9],[15,7]]`

## Example 2:
## *Ví dụ 2:*

**Input:** `root = [1]`
**Output:** `[[1]]`

## Example 3:
## *Ví dụ 3:*

**Input:** `root = []`
**Output:** `[]`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the tree is in the range `[0, 2000]`.
*   `-100 <= Node.val <= 100`
