# Result for Intersection of Two Arrays
# *Kết quả cho bài toán Giao của Hai Mảng*

## Description
## *Mô tả*

Given two integer arrays `nums1` and `nums2`, return *an array of their intersection*. Each element in the result must be **unique** and you may return the result in **any order**.
*Cho hai mảng số nguyên `nums1` và `nums2`, trả về *một mảng chứa phần giao của chúng*. Mỗi phần tử trong kết quả phải là **duy nhất** và bạn có thể trả về kết quả theo **bất kỳ thứ tự nào**.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums1 = [1,2,2,1], nums2 = [2,2]`
**Output:** `[2]`

## Example 2:
## *Ví dụ 2:*

**Input:** `nums1 = [4,9,5], nums2 = [9,4,9,8,4]`
**Output:** `[9,4]`
**Explanation:** [4,9] is also accepted.

## Constraints:
## *Ràng buộc:*

*   `1 <= nums1.length, nums2.length <= 1000`
*   `0 <= nums1[i], nums2[i] <= 1000`
