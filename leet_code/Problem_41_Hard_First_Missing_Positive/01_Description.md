# Result for First Missing Positive
# *Kết quả cho bài toán Số Dương Đầu Tiên Còn Thiếu*

## Description
## *Mô tả*

Given an unsorted integer array `nums`, return the smallest missing positive integer.
*Cho một mảng số nguyên chưa được sắp xếp `nums`, hãy trả về số nguyên dương nhỏ nhất còn thiếu.*

You must implement an algorithm that runs in `O(n)` time and uses `O(1)` auxiliary space.
*Bạn phải triển khai một thuật toán chạy trong thời gian `O(n)` và sử dụng không gian phụ trợ `O(1)`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [1,2,0]`
**Output:** `3`
**Explanation:** The numbers in the range [1,2] are all in the array.

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [3,4,-1,1]`
**Output:** `2`
**Explanation:** 1 is in the array but 2 is missing.

## Example 3:
## *Ví dụ 3:*

**Input:** `nums = [7,8,9,11,12]`
**Output:** `1`
**Explanation:** The smallest positive integer 1 is missing.

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 10^5`
*   `-2^31 <= nums[i] <= 2^31 - 1`
