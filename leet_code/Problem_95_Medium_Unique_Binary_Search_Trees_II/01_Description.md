# Result for Unique Binary Search Trees II
# *Kết quả cho bài toán Cây Tìm Kiếm Nhị Phân Duy Nhất II*

## Description
## *Mô tả*

Given an integer `n`, return *all the structurally unique **BST's** (binary search trees), which has exactly `n` nodes of unique values from `1` to `n`*. Return the answer in **any order**.
*Cho một số nguyên `n`, hãy trả về *tất cả các **BST** (cây tìm kiếm nhị phân) có cấu trúc duy nhất, có chính xác `n` nút với các giá trị duy nhất từ `1` đến `n`*. Trả về câu trả lời theo **bất kỳ thứ tự nào**.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 3`
**Output:** `[[1,null,2,null,3],[1,null,3,2],[2,1,3],[3,1,null,null,2],[3,2,null,1]]`

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 1`
**Output:** `[[1]]`

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 8`
