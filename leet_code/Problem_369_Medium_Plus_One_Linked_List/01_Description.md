# Result for Plus One Linked List
# *Kết quả cho bài toán Cộng Một vào Danh sách Liên kết*

## Description
## *Mô tả*

Given a non-negative integer represented as a linked list of digits, *plus one to the integer*.
*Cho một số nguyên không âm được đại diện dưới dạng một danh sách liên kết các chữ số, hãy *cộng thêm một vào số nguyên đó*.*

The digits are stored such that the most significant digit is at the **head** of the list.
*Các chữ số được lưu trữ sao cho chữ số quan trọng nhất (hàng cao nhất) nằm ở **đầu (head)** của danh sách.*

## Example 1:
## *Ví dụ 1:*

**Input:** `head = [1,2,3]`
**Output:** `[1,2,4]`

## Example 2:
## *Ví dụ 2:*

**Input:** `head = [0]`
**Output:** `[1]`

## Example 3:
## *Ví dụ 3:*

**Input:** `head = [9,9,9]`
**Output:** `[1,0,0,0]`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the linked list is in the range `[1, 100]`.
*   `0 <= Node.val <= 9`
*   The number represented by the linked list does not contain leading zeros except for the zero itself.
