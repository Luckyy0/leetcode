# Result for Next Greater Element III
# *Kết quả cho bài toán Phần tử Lớn hơn Tiếp theo III*

## Description
## *Mô tả*

Given a positive integer `n`, find the smallest integer which has exactly the same digits existing in the integer `n` and is greater in value than `n`. If no such positive integer exists, return `-1`.
*Cho một số nguyên dương `n`, hãy tìm số nguyên nhỏ nhất có các chữ số giống hệt với số nguyên `n` và có giá trị lớn hơn `n`. Nếu không tồn tại số nguyên dương như vậy, hãy trả về `-1`.*

Note that the returned integer should fit in **32-bit integer**, if there is a valid answer but it does not fit in **32-bit integer**, return `-1`.
*Lưu ý rằng số nguyên được trả về phải nằm trong **số nguyên 32 bit**, nếu có câu trả lời hợp lệ nhưng nó không nằm trong **số nguyên 32 bit**, hãy trả về `-1`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 12`
**Output:** `21`

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 21`
**Output:** `-1`

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 2^31 - 1`
