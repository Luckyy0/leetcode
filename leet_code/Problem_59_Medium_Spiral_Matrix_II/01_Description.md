# Result for Spiral Matrix II
# *Kết quả cho bài toán Ma Trận Xoắn Ốc II*

## Description
## *Mô tả*

Given a positive integer `n`, generate an `n x n` `matrix` filled with elements from `1` to `n^2` in spiral order.
*Cho một số nguyên dương `n`, hãy tạo một ma trận `n x n` được điền các phần tử từ `1` đến `n^2` theo thứ tự xoắn ốc.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 3`
**Output:** `[[1,2,3],[8,9,4],[7,6,5]]`

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 1`
**Output:** `[[1]]`

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 20`
