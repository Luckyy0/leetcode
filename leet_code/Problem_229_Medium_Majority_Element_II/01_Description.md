# Result for Majority Element II
# *Kết quả cho bài toán Phần tử đa số II*

## Description
## *Mô tả*

Given an integer array of size `n`, find all elements that appear more than `⌊ n/3 ⌋` times.
*Cho một mảng số nguyên có kích thước `n`, hãy tìm tất cả các phần tử xuất hiện nhiều hơn `⌊ n/3 ⌋` lần.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [3,2,3]`
**Output:** `[3]`

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [1]`
**Output:** `[1]`

## Example 3:
## *Ví dụ 3:*

**Input:** `nums = [1,2]`
**Output:** `[1,2]`

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 5 * 10^4`
*   `-10^9 <= nums[i] <= 10^9`

**Follow up:** Could you solve the problem in linear time and in `O(1)` space?
**Câu hỏi mở rộng:** Bạn có thể giải quyết bài toán trong thời gian tuyến tính và không gian `O(1)` không?
