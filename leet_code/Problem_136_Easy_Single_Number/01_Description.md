# Result for Single Number
# *Kết quả cho bài toán Số Duy Nhất*

## Description
## *Mô tả*

Given a **non-empty** array of integers `nums`, every element appears *twice* except for one. Find that single one.
*Cho một mảng số nguyên `nums` **không trống**, mọi phần tử đều xuất hiện *hai lần* ngoại trừ một phần tử. Hãy tìm phần tử duy nhất đó.*

You must implement a solution with a linear runtime complexity and use only constant extra space.
*Bạn phải triển khai một giải pháp với độ phức tạp thời gian tuyến tính và chỉ sử dụng không gian bổ sung hằng số.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [2,2,1]`
**Output:** `1`

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [4,1,2,1,2]`
**Output:** `4`

## Example 3:
## *Ví dụ 3:*

**Input:** `nums = [1]`
**Output:** `1`

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 3 * 10^4`
*   `-3 * 10^4 <= nums[i] <= 3 * 10^4`
*   Each element in the array appears twice except for one element which appears only once.
