# Result for Combinations
# *Kết quả cho bài toán Tổ Hợp*

## Description
## *Mô tả*

Given two integers `n` and `k`, return *all possible combinations of `k` numbers chosen from the range `[1, n]`*.
*Cho hai số nguyên `n` và `k`, hãy trả về *tất cả các tổ hợp có thể có của `k` số được chọn từ khoảng `[1, n]`**.*

You may return the answer in **any order**.
*Bạn có thể trả về câu trả lời theo **bất kỳ thứ tự nào**.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 4, k = 2`
**Output:** `[[1,2],[1,3],[1,4],[2,3],[2,4],[3,4]]`
**Explanation:** There are 4C2 = 6 combinations.

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 1, k = 1`
**Output:** `[[1]]`
**Explanation:** There is 1C1 = 1 combination.

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 20`
*   `1 <= k <= n`
