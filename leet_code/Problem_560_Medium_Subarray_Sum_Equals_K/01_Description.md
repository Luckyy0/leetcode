# Result for Subarray Sum Equals K
# *Kết quả cho bài toán Tổng Mảng con bằng K*

## Description
## *Mô tả*

Given an array of integers `nums` and an integer `k`, return *the total number of subarrays whose sum equals to* `k`.
*Cho một mảng các số nguyên `nums` và một số nguyên `k`, hãy trả về *tổng số mảng con có tổng bằng* `k`.*

A subarray is a contiguous **non-empty** sequence of elements within an array.
*Một mảng con là một chuỗi các phần tử liên tiếp **không rỗng** trong một mảng.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [1,1,1], k = 2`
**Output:** `2`

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [1,2,3], k = 3`
**Output:** `2`

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 2 * 10^4`
*   `-1000 <= nums[i] <= 1000`
*   `-10^7 <= k <= 10^7`
