# Result for Reverse Linked List
# *Kết quả cho bài toán Đảo ngược danh sách liên kết*

## Description
## *Mô tả*

Given the `head` of a singly linked list, reverse the list, and return *the reversed list*.
*Cho `head` của một danh sách liên kết đơn, hãy đảo ngược danh sách và trả về *danh sách đã đảo ngược***.*

## Example 1:
## *Ví dụ 1:*

**Input:** `head = [1,2,3,4,5]`
**Output:** `[5,4,3,2,1]`

## Example 2:
## *Ví dụ 2:*

**Input:** `head = [1,2]`
**Output:** `[2,1]`

## Example 3:
## *Ví dụ 3:*

**Input:** `head = []`
**Output:** `[]`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the list is the range `[0, 5000]`.
*   `-5000 <= Node.val <= 5000`

**Follow up:** A linked list can be reversed either iteratively or recursively. Could you implement both?
**Câu hỏi mở rộng:** Một danh sách liên kết có thể được đảo ngược theo cách lặp hoặc đệ quy. Bạn có thể triển khai cả hai không?
