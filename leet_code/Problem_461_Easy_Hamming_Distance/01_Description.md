# Result for Hamming Distance
# *Kết quả cho bài toán Khoảng cách Hamming*

## Description
## *Mô tả*

The [Hamming distance](https://en.wikipedia.org/wiki/Hamming_distance) between two integers is the number of positions at which the corresponding bits are different.
*Khoảng cách Hamming giữa hai số nguyên là số vị trí mà các bit tương ứng tại đó khác nhau.*

Given two integers `x` and `y`, return *the **Hamming distance** between them*.
*Cho hai số nguyên `x` và `y`, hãy trả về *khoảng cách Hamming giữa chúng*.*

## Example 1:
## *Ví dụ 1:*

**Input:** `x = 1, y = 4`
**Output:** `2`
**Explanation:**
1   (0 0 0 1)
4   (0 1 0 0)
       ↑   ↑
The above arrows point to positions where the corresponding bits are different.

## Example 2:
## *Ví dụ 2:*

**Input:** `x = 3, y = 1`
**Output:** `1`

## Constraints:
## *Ràng buộc:*

*   `0 <= x, y <= 2^{31} - 1`
