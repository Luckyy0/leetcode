# Result for Single Number II
# *Kết quả cho bài toán Số Duy Nhất II*

## Description
## *Mô tả*

Given an integer array `nums` where every element appears **three times** except for one, which appears **exactly once**. Find the single element and return it.
*Cho một mảng số nguyên `nums` trong đó mọi phần tử đều xuất hiện **ba lần** ngoại trừ một phần tử, phần tử này xuất hiện đúng **một lần**. Hãy tìm phần tử duy nhất đó và trả về.*

You must implement a solution with a linear runtime complexity and use only constant extra space.
*Bạn phải triển khai một giải pháp với độ phức tạp thời gian tuyến tính và chỉ sử dụng không gian bổ sung hằng số.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [2,2,3,2]`
**Output:** `3`

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [0,1,0,1,0,1,99]`
**Output:** `99`

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 3 * 10^4`
*   `-2^31 <= nums[i] <= 2^31 - 1`
*   Each element in `nums` appears exactly three times except for one element which appears once.
