# Result for Reorganize String
# *Kết quả cho bài toán Tái cấu trúc Chuỗi*

## Description
## *Mô tả*

Given a string `s`, rearrange the characters of `s` so that any two adjacent characters are not the same.
*Cho một chuỗi `s`, hãy sắp xếp lại các ký tự của `s` sao cho bất kỳ hai ký tự kề nhau nào cũng không giống nhau.*

Return *any possible rearrangement of `s` or return `""` if not possible*.
*Trả về *bất kỳ cách sắp xếp nào có thể của `s` hoặc trả về `""` nếu không thể*.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** s = "aab"
**Output:** "aba"

## Example 2:
## *Ví dụ 2:*

**Input:** s = "aaab"
**Output:** ""

---

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 500`
*   `s` consists of lowercase English letters.
