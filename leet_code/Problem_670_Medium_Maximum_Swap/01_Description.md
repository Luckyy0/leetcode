# Result for Maximum Swap
# *Kết quả cho bài toán Hoán đổi Tối đa*

## Description
## *Mô tả*

You are given an integer `num`. You can swap two digits at most once to get the maximum valued number.
*Cho một số nguyên `num`. Bạn có thể hoán đổi hai chữ số tối đa một lần để có được số có giá trị lớn nhất.*

Return the maximum valued number you can get.
*Hãy trả về số có giá trị lớn nhất mà bạn có thể nhận được.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** num = 2736
**Output:** 7236
**Explanation:** Swap 2 and 7.

## Example 2:
## *Ví dụ 2:*

**Input:** num = 9973
**Output:** 9973
**Explanation:** No swap.

---

## Constraints:
## *Ràng buộc:*

*   `0 <= num <= 10^8`
