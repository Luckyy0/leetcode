# Result for Subsets II
# *Kết quả cho bài toán Tập Con II*

## Description
## *Mô tả*

Given an integer array `nums` that may contain duplicates, return *all possible subsets (the power set)*.
*Cho một mảng số nguyên `nums` có thể chứa các phần tử trùng lặp, hãy trả về *tất cả các tập con có thể có (tập hợp lũy thừa)***.*

The solution set **must not** contain duplicate subsets. Return the solution in **any order**.
*Tập hợp giải pháp **không được** chứa các tập con trùng lặp. Trả về giải pháp theo **bất kỳ thứ tự nào**.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [1,2,2]`
**Output:** `[[],[1],[1,2],[1,2,2],[2],[2,2]]`

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [0]`
**Output:** `[[],[0]]`

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 10`
*   `-10 <= nums[i] <= 10`
