# Result for Remove Nth Node From End of List
# *Kết quả cho bài toán Xóa Nút Thứ N Từ Cuối Danh Sách*

## Description
## *Mô tả*

Given the `head` of a linked list, remove the `n-th` node from the end of the list and return its head.
*Cho `head` của một danh sách liên kết, hãy xóa nút thứ `n` từ cuối danh sách và trả về đầu danh sách.*

## Example 1:
## *Ví dụ 1:*

**Input:** `head = [1,2,3,4,5]`, `n = 2`
**Output:** `[1,2,3,5]`
**Explanation:** The 2nd node from the end is 4. Removing it yield [1,2,3,5].
***Giải thích:** Nút thứ 2 từ cuối là 4. Xóa nó sẽ được [1,2,3,5].*

## Example 2:
## *Ví dụ 2:*

**Input:** `head = [1]`, `n = 1`
**Output:** `[]`

## Example 3:
## *Ví dụ 3:*

**Input:** `head = [1,2]`, `n = 1`
**Output:** `[1]`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the list is `sz`.
*   `1 <= sz <= 30`
*   `0 <= Node.val <= 100`
*   `1 <= n <= sz`
