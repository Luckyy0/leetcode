# Result for Balanced Binary Tree
# *Kết quả cho bài toán Cây Nhị Phân Cân Bằng*

## Description
## *Mô tả*

Given a binary tree, determine if it is **height-balanced**.
*Cho một cây nhị phân, hãy xác định xem nó có **cân bằng về chiều cao** hay không.*

A **height-balanced** binary tree is defined as:
*Một cây nhị phân **cân bằng về chiều cao** được định nghĩa là:*

> a binary tree in which the left and right subtrees of every node differ in height by no more than 1.

## Example 1:
## *Ví dụ 1:*

**Input:** `root = [3,9,20,null,null,15,7]`
**Output:** `true`

## Example 2:
## *Ví dụ 2:*

**Input:** `root = [1,2,2,3,3,null,null,4,4]`
**Output:** `false`

## Example 3:
## *Ví dụ 3:*

**Input:** `root = []`
**Output:** `true`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the tree is in the range `[0, 5000]`.
*   `-10^4 <= Node.val <= 10^4`
