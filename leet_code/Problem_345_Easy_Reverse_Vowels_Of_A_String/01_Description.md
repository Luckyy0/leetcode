# Result for Reverse Vowels of a String
# *Kết quả cho bài toán Đảo ngược Nguyên âm của một Chuỗi*

## Description
## *Mô tả*

Given a string `s`, reverse only all the vowels in the string and return it.
*Cho một chuỗi `s`, chỉ đảo ngược tất cả các nguyên âm trong chuỗi và trả về nó.*

The vowels are `'a'`, `'e'`, `'i'`, `'o'`, and `'u'`, and they can appear in both lower and upper cases, more than once.
*Các nguyên âm là `'a'`, `'e'`, `'i'`, `'o'`, và `'u'`, và chúng có thể xuất hiện ở cả chữ thường và chữ hoa, nhiều lần.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = "hello"`
**Output:** `holle`

## Example 2:
## *Ví dụ 2:*

**Input:** `s = "leetcode"`
**Output:** `leotcede`

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 3 * 10^5`
*   `s` consist of printable ASCII characters.
