# Result for Longest Palindromic Substring
# *Kết quả cho bài toán Chuỗi Con Đối Xứng Dài Nhất*

## Description
## *Mô tả*

Given a string `s`, return the **longest palindromic substring** in `s`.
*Cho một chuỗi `s`, trả về **chuỗi con đối xứng dài nhất** trong `s`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = "babad"`
**Output:** `"bab"`
**Explanation:** "aba" is also a valid answer.
***Giải thích:** "aba" cũng là một câu trả lời hợp lệ.*

## Example 2:
## *Ví dụ 2:*

**Input:** `s = "cbbd"`
**Output:** `"bb"`

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 1000`
*   `s` consist of only digits and English letters.
*   *`s` chỉ bao gồm các chữ số và chữ cái tiếng Anh.*
