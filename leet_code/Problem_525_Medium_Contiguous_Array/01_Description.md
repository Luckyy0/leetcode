# Result for Contiguous Array
# *Kết quả cho bài toán Mảng Liên tục*

## Description
## *Mô tả*

Given a binary array `nums`, return *the maximum length of a contiguous subarray with an equal number of* `0` *and* `1`.
*Cho một mảng nhị phân `nums`, hãy trả về *độ dài lớn nhất của một mảng con liên tục có số lượng* `0` *và* `1` *bằng nhau*.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [0,1]`
**Output:** `2`
**Explanation:** [0, 1] is the longest contiguous subarray with an equal number of 0 and 1.

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [0,1,0]`
**Output:** `2`
**Explanation:** [0, 1] (or [1, 0]) is a longest contiguous subarray with equal number of 0 and 1.

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 10^5`
*   `nums[i]` is either `0` or `1`.
