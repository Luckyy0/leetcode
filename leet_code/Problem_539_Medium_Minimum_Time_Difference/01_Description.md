# Result for Minimum Time Difference
# *Kết quả cho bài toán Hiệu Thời gian Nhỏ nhất*

## Description
## *Mô tả*

Given a list of 24-hour clock time points in **"HH:MM"** format, return *the minimum **minutes** difference between any two time-points in the list*.
*Cho một danh sách các điểm thời gian theo đồng hồ 24 giờ ở định dạng **"HH:MM"**, trả về *hiệu số **phút** nhỏ nhất giữa bất kỳ hai điểm thời gian nào trong danh sách*.*

## Example 1:
## *Ví dụ 1:*

**Input:** `timePoints = ["23:59","00:00"]`
**Output:** `1`

## Example 2:
## *Ví dụ 2:*

**Input:** `timePoints = ["00:00","23:59","00:00"]`
**Output:** `0`

## Constraints:
## *Ràng buộc:*

*   `2 <= timePoints.length <= 2 * 10^4`
*   `timePoints[i]` is in the format **"HH:MM"**.
