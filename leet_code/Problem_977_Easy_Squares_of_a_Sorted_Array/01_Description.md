# Result for Squares of a Sorted Array
# *Kết quả cho bài toán Bình phương của Mảng đã Sắp xếp*

## Description
## *Mô tả*

Given an integer array `nums` sorted in **non-decreasing** order, return *an array of the squares of each number sorted in non-decreasing order*.
*Cho một mảng số nguyên `nums` được sắp xếp theo thứ tự **không giảm**, trả về *một mảng các bình phương của mỗi số được sắp xếp theo thứ tự không giảm*.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** nums = [-4,-1,0,3,10]
**Output:** [0,1,9,16,100]
**Explanation:** After squaring, the array becomes [16,1,0,9,100].
After sorting, it becomes [0,1,9,16,100].

## Example 2:
## *Ví dụ 2:*

**Input:** nums = [-7,-3,2,3,11]
**Output:** [4,9,9,49,121]

---

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 10^4`
*   `-10^4 <= nums[i] <= 10^4`
*   `nums` is sorted in **non-decreasing** order.
