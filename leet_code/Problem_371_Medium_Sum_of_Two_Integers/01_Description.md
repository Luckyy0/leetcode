# Result for Sum of Two Integers
# *Kết quả cho bài toán Tổng của Hai số nguyên*

## Description
## *Mô tả*

Given two integers `a` and `b`, return the sum of the two integers without using the operators `+` and `-`.
*Cho hai số nguyên `a` và `b`, hãy trả về tổng của hai số nguyên đó mà không sử dụng các toán tử `+` và `-`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `a = 1, b = 2`
**Output:** `3`

## Example 2:
## *Ví dụ 2:*

**Input:** `a = 2, b = 3`
**Output:** `5`

## Constraints:
## *Ràng buộc:*

*   `-1000 <= a, b <= 1000`
