# Result for Find the Difference
# *Kết quả cho bài toán Tìm sự Khác biệt*

## Description
## *Mô tả*

You are given two strings `s` and `t`.
*Bạn được cho hai chuỗi `s` và `t`.*

String `t` is generated by random shuffling string `s` and then add one more letter at a random position.
*Chuỗi `t` được tạo ra bằng cách xáo trộn ngẫu nhiên chuỗi `s` và sau đó thêm một chữ cái nữa vào một vị trí ngẫu nhiên.*

Return *the letter that was added to `t`*.
*Trả về *chữ cái đã được thêm vào `t`*.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = "abcd", t = "abcde"`
**Output:** `"e"`
**Explanation:** 'e' is the letter that was added.

## Example 2:
## *Ví dụ 2:*

**Input:** `s = "", t = "y"`
**Output:** `"y"`

## Constraints:
## *Ràng buộc:*

*   `0 <= s.length <= 1000`
*   `t.length == s.length + 1`
*   `s` and `t` consist of lowercase English letters.
