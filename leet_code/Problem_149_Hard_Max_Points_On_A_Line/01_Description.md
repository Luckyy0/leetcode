# Result for Max Points on a Line
# *Kết quả cho bài toán Số điểm tối đa trên một đường thẳng*

## Description
## *Mô tả*

Given an array of `points` where `points[i] = [xi, yi]` represents a point on the **X-Y** plane, return *the maximum number of points that lie on the same straight line*.
*Cho một mảng các `points` trong đó `points[i] = [xi, yi]` đại diện cho một điểm trên mặt phẳng **X-Y**, hãy trả về *số lượng điểm tối đa nằm trên cùng một đường thẳng***.*

## Example 1:
## *Ví dụ 1:*

**Input:** `points = [[1,1],[2,2],[3,3]]`
**Output:** `3`

## Example 2:
## *Ví dụ 2:*

**Input:** `points = [[1,1],[3,2],[5,3],[4,1],[2,3],[1,4]]`
**Output:** `4`

## Constraints:
## *Ràng buộc:*

*   `1 <= points.length <= 300`
*   `points[i].length == 2`
*   `-10^4 <= xi, yi <= 10^4`
*   All the `points` are **unique**.
