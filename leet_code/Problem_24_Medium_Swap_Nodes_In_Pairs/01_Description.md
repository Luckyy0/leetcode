# Result for Swap Nodes in Pairs
# *Kết quả cho bài toán Hoán Đổi Các Nút Theo Cặp*

## Description
## *Mô tả*

Given a linked list, swap every two adjacent nodes and return its head.
*Cho một danh sách liên kết, hoán đổi hai nút liền kề nhau và trả về đầu danh sách.*

You must solve the problem without modifying the values in the list's nodes (i.e., only nodes themselves may be changed.)
*Bạn phải giải quyết bài toán mà không thay đổi giá trị trong các nút của danh sách (nghĩa là, chỉ các nút mới được thay đổi).*

## Example 1:
## *Ví dụ 1:*

**Input:** `head = [1,2,3,4]`
**Output:** `[2,1,4,3]`

## Example 2:
## *Ví dụ 2:*

**Input:** `head = []`
**Output:** `[]`

## Example 3:
## *Ví dụ 3:*

**Input:** `head = [1]`
**Output:** `[1]`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the list is in the range `[0, 100]`.
*   `0 <= Node.val <= 100`
