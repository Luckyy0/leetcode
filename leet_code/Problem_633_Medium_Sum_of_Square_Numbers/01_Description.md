# Result for Sum of Square Numbers
# *Kết quả cho bài toán Tổng của hai Số Chính phương*

## Description
## *Mô tả*

Given a non-negative integer `c`, decide whether there are two integers `a` and `b` such that `a^2 + b^2 = c`.
*Cho một số nguyên không âm `c`, hãy quyết định xem có tồn tại hai số nguyên `a` và `b` sao cho `a^2 + b^2 = c` hay không.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** c = 5
**Output:** true
**Explanation:** 1 * 1 + 2 * 2 = 5

## Example 2:
## *Ví dụ 2:*

**Input:** c = 3
**Output:** false

---

## Constraints:
## *Ràng buộc:*

*   `0 <= c <= 2^31 - 1`
