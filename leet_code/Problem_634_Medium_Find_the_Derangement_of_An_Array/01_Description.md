# Result for Find the Derangement of An Array
# *Kết quả cho bài toán Tìm số Hoán vị Không cố định của một Mảng*

## Description
## *Mô tả*

In combinatorial mathematics, a **derangement** is a permutation of the elements of a set, in which no element appears in its original position.
*Trong toán học tổ hợp, một **hoán vị không cố định** (derangement) là một hoán vị của các phần tử trong một tập hợp, trong đó không có phần tử nào xuất hiện ở vị trí ban đầu của nó.*

You are given an integer `n`, return the number of derangements of an array of `n` elements. Since the answer may be very large, return it **modulo** `10^9 + 7`.
*Cho một số nguyên `n`, hãy trả về số lượng các hoán vị không cố định của một mảng có `n` phần tử. Vì câu trả lời có thể rất lớn, hãy trả về kết quả theo **modulo** `10^9 + 7`.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** n = 3
**Output:** 2
**Explanation:** The original array is [1,2,3]. The two derangements are [2,3,1] and [3,1,2].

## Example 2:
## *Ví dụ 2:*

**Input:** n = 2
**Output:** 1

---

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 10^6`
