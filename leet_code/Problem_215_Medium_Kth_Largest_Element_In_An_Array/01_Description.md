# Result for Kth Largest Element in an Array
# *Kết quả cho bài toán Phần tử lớn thứ K trong mảng*

## Description
## *Mô tả*

Given an integer array `nums` and an integer `k`, return *the* `k`*th largest element in the array*.
*Cho một mảng số nguyên `nums` và một số nguyên `k`, hãy trả về *phần tử lớn thứ `k` trong mảng***.*

Note that it is the `k`th largest element in the sorted order, not the `k`th distinct element.
*Lưu ý rằng đây là phần tử lớn thứ `k` theo thứ tự sắp xếp, không phải phần tử khác biệt thứ `k`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [3,2,1,5,6,4], k = 2`
**Output:** `5`

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [3,2,3,1,2,4,5,5,6], k = 4`
**Output:** `4`

## Constraints:
## *Ràng buộc:*

*   `1 <= k <= nums.length <= 10^5`
*   `-10^4 <= nums[i] <= 10^4`
