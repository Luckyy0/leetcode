# Result for Valid Triangle Number
# *Kết quả cho bài toán Số lượng Tam giác Hợp lệ*

## Description
## *Mô tả*

Given an integer array `nums`, return the number of triplets chosen from the array that can make triangles if we take them as side lengths of a triangle.
*Cho một mảng số nguyên `nums`, hãy trả về số lượng bộ ba được chọn từ mảng có thể tạo thành hình tam giác nếu chúng ta lấy chúng làm độ dài các cạnh của một hình tam giác.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** nums = [2,2,3,4]
**Output:** 3
**Explanation:** Valid combinations are: 
2,3,4 (using the first 2)
2,3,4 (using the second 2)
2,2,3

## Example 2:
## *Ví dụ 2:*

**Input:** nums = [4,2,3,4]
**Output:** 4

---

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 1000`
*   `0 <= nums[i] <= 1000`
