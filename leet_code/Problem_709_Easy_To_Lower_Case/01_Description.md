# Result for To Lower Case
# *Kết quả cho bài toán Chuyển thành chữ thường*

## Description
## *Mô tả*

Given a string `s`, return the string after replacing every uppercase letter with the same lowercase letter.
*Cho một chuỗi `s`, trả về chuỗi sau khi thay thế mọi chữ cái viết hoa bằng chữ cái viết thường tương ứng.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** s = "Hello"
**Output:** "hello"

## Example 2:
## *Ví dụ 2:*

**Input:** s = "here"
**Output:** "here"

## Example 3:
## *Ví dụ 3:*

**Input:** s = "LOVELY"
**Output:** "lovely"

---

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 100`
*   `s` consists of printable ASCII characters.
