# Result for Symmetric Tree
# *Kết quả cho bài toán Cây Đối Xứng*

## Description
## *Mô tả*

Given the `root` of a binary tree, *check whether it is a mirror of itself* (i.e., symmetric around its center).
*Cho `root` của một cây nhị phân, hãy *kiểm tra xem nó có phải là bản sao gương của chính nó hay không* (tức là đối xứng qua tâm của nó).*

## Example 1:
## *Ví dụ 1:*

**Input:** `root = [1,2,2,3,4,4,3]`
**Output:** `true`

## Example 2:
## *Ví dụ 2:*

**Input:** `root = [1,2,2,null,3,null,3]`
**Output:** `false`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the tree is in the range `[1, 1000]`.
*   `-100 <= Node.val <= 100`

## Follow up:
*   Could you solve it both recursively and iteratively?
