# Result for Base 7
# *Kết quả cho bài toán Hệ cơ số 7*

## Description
## *Mô tả*

Given an integer `num`, return *a string of its **base 7** representation*.
*Cho một số nguyên `num`, hãy trả về *một chuỗi biểu diễn **hệ cơ số 7** của nó*.*

## Example 1:
## *Ví dụ 1:*

**Input:** `num = 100`
**Output:** `"202"`

## Example 2:
## *Ví dụ 2:*

**Input:** `num = -7`
**Output:** `"-10"`

## Constraints:
## *Ràng buộc:*

*   `-10^7 <= num <= 10^7`
