# Result for Add Strings
# *Kết quả cho bài toán Cộng Chuỗi*

## Description
## *Mô tả*

Given two non-negative integers, `num1` and `num2` represented as string, return *the sum of `num1` and `num2` as a string*.
*Cho hai số nguyên không âm, `num1` và `num2` được đại diện dưới dạng chuỗi, hãy trả về *tổng của `num1` và `num2` dưới dạng một chuỗi*.*

You must solve the problem without using any built-in library for handling large integers (such as `BigInteger`). You must also not convert the inputs to integers directly.
*Bạn phải giải bài toán mà không sử dụng bất kỳ thư viện có sẵn nào để xử lý các số nguyên lớn (như `BigInteger`). Bạn cũng không được chuyển đổi trực tiếp đầu vào thành số nguyên.*

## Example 1:
## *Ví dụ 1:*

**Input:** `num1 = "11", num2 = "123"`
**Output:** `"134"`

## Example 2:
## *Ví dụ 2:*

**Input:** `num1 = "456", num2 = "77"`
**Output:** `"533"`

## Example 3:
## *Ví dụ 3:*

**Input:** `num1 = "0", num2 = "0"`
**Output:** `"0"`

## Constraints:
## *Ràng buộc:*

*   `1 <= num1.length, num2.length <= 10^4`
*   `num1` and `num2` consist of only digits.
*   `num1` and `num2` don't have any leading zeros except for the zero itself.
