# Result for Maximum Product of Three Numbers
# *Kết quả cho bài toán Tích Lớn nhất của Ba số*

## Description
## *Mô tả*

Given an integer array `nums`, find three numbers whose product is maximum and return the maximum product.
*Cho một mảng số nguyên `nums`, hãy tìm ba số có tích lớn nhất và trả về giá trị tích lớn nhất đó.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** nums = [1,2,3]
**Output:** 6

## Example 2:
## *Ví dụ 2:*

**Input:** nums = [1,2,3,4]
**Output:** 24

## Example 3:
## *Ví dụ 3:*

**Input:** nums = [-1,-2,-3]
**Output:** -6

---

## Constraints:
## *Ràng buộc:*

*   `3 <= nums.length <= 10^4`
*   `-1000 <= nums[i] <= 1000`
