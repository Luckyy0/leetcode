# Result for Minimum Distance Between BST Nodes
# *Kết quả cho bài toán Khoảng cách Tối thiểu Giữa các Nút BST*

## Description
## *Mô tả*

Given the `root` of a Binary Search Tree (BST), return *the minimum difference between the values of any two different nodes in the tree*.
*Cho `root` của một Cây Tìm kiếm Nhị phân (BST), trả về *giá trị chênh lệch tối thiểu giữa các giá trị của bất kỳ hai nút khác nhau nào trong cây*.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** root = [4,2,6,1,3]
**Output:** 1
**Explanation:**
The distinct values are 1, 2, 3, 4, 6.
The minimum difference is 1 (between 1 and 2, 2 and 3, 3 and 4).

## Example 2:
## *Ví dụ 2:*

**Input:** root = [1,0,48,null,null,12,49]
**Output:** 1

---

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the tree is in the range `[2, 100]`.
*   `0 <= Node.val <= 10^5`
