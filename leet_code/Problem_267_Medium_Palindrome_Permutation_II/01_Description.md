# Result for Palindrome Permutation II
# *Kết quả cho bài toán Hoán vị Palindrome II*

## Description
## *Mô tả*

Given a string `s`, return *all the palindromic permutations (without duplicates) of it*.
*Cho một chuỗi `s`, trả về *tất cả các hoán vị palindrome (không trùng lặp) của nó*.*

You may return the answer in **any order**. If `s` has no palindromic permutation, return an empty list.
*Bạn có thể trả về câu trả lời theo **bất kỳ thứ tự nào**. Nếu `s` không có hoán vị palindrome, trả về danh sách rỗng.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = "aabb"`
**Output:** `["abba","baab"]`

## Example 2:
## *Ví dụ 2:*

**Input:** `s = "abc"`
**Output:** `[]`

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 16`
*   `s` consists of only lowercase English letters.
