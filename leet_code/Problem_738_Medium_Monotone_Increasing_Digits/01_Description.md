# Result for Monotone Increasing Digits
# *Kết quả cho bài toán Số có các Chữ số Tăng dần Đơn điệu (Monotone Increasing Digits)*

## Description
## *Mô tả*

An integer has **monotone increasing digits** if and only if each pair of adjacent digits `x` and `y` satisfy `x <= y`.
*Một số nguyên có **các chữ số tăng dần đơn điệu** khi và chỉ khi mỗi cặp chữ số liền kề `x` và `y` thỏa mãn `x <= y`.*

Given an integer `n`, return the largest integer less than or equal to `n` with monotone increasing digits.
*Cho một số nguyên `n`, hãy trả về số nguyên lớn nhất nhỏ hơn hoặc bằng `n` có các chữ số tăng dần đơn điệu.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** n = 10
**Output:** 9

## Example 2:
## *Ví dụ 2:*

**Input:** n = 1234
**Output:** 1234

## Example 3:
## *Ví dụ 3:*

**Input:** n = 332
**Output:** 299

---

## Constraints:
## *Ràng buộc:*

*   `0 <= n <= 10^9`
