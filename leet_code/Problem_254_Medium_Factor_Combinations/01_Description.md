# Result for Factor Combinations
# *Kết quả cho bài toán Tổ hợp thừa số*

## Description
## *Mô tả*

Numbers can be regarded as product of its factors. For example,
*Các số có thể được coi là tích của các thừa số của nó. Ví dụ:*

`8 = 2 x 2 x 2`
` = 2 x 4`

Given an integer `n`, return *all possible combinations of its factors*. You may return the answer in **any order**.
*Cho một số nguyên `n`, hãy trả về *tất cả các tổ hợp có thể có của các thừa số của nó*. Bạn có thể trả về câu trả lời theo **bất kỳ thứ tự nào**.*

**Note** that the factors should be in the range `[2, n - 1]`.
*Lưu ý rằng các thừa số nên nằm trong phạm vi `[2, n - 1]`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 1`
**Output:** `[]`

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 12`
**Output:** `[[2,6],[2,2,3],[3,4]]`

## Example 3:
## *Ví dụ 3:*

**Input:** `n = 37`
**Output:** `[]`

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 10^7`
