# Result for Power of Four
# *Kết quả cho bài toán Lũy thừa của Bốn*

## Description
## *Mô tả*

Given an integer `n`, return `true` if it is a power of four. Otherwise, return `false`.
*Cho một số nguyên `n`, trả về `true` nếu nó là lũy thừa của bốn. Ngược lại, trả về `false`.*

An integer `n` is a power of four, if there exists an integer `x` such that `n == 4^x`.
*Một số nguyên `n` là lũy thừa của bốn, nếu tồn tại một số nguyên `x` sao cho `n == 4^x`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 16`
**Output:** `true`

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 5`
**Output:** `false`

## Example 3:
## *Ví dụ 3:*

**Input:** `n = 1`
**Output:** `true`

## Constraints:
## *Ràng buộc:*

*   `-2^31 <= n <= 2^31 - 1`

**Follow up:** Could you solve it without loops/recursion?
**Câu hỏi mở rộng:** Bạn có thể giải quyết nó mà không cần vòng lặp/đệ quy không?
