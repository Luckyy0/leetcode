# Result for Count Primes
# *Kết quả cho bài toán Đếm số nguyên tố*

## Description
## *Mô tả*

Given an integer `n`, return *the number of prime numbers that are strictly less than `n`*.
*Cho một số nguyên `n`, hãy trả về *số lượng số nguyên tố nhỏ hơn hoàn toàn số `n`***.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 10`
**Output:** `4`
**Explanation:** There are 4 prime numbers less than 10, they are 2, 3, 5, 7.

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 0`
**Output:** `0`

## Example 3:
## *Ví dụ 3:*

**Input:** `n = 1`
**Output:** `0`

## Constraints:
## *Ràng buộc:*

*   `0 <= n <= 5 * 10^6`
