# Result for Maximum Subarray
# *Kết quả cho bài toán Mảng Con Lớn Nhất*

## Description
## *Mô tả*

Given an integer array `nums`, find the subarray with the largest sum, and return its sum.
*Cho một mảng số nguyên `nums`, hãy tìm mảng con có tổng lớn nhất và trả về tổng của nó.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [-2,1,-3,4,-1,2,1,-5,4]`
**Output:** `6`
**Explanation:** The subarray `[4,-1,2,1]` has the largest sum 6.

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [1]`
**Output:** `1`
**Explanation:** The subarray `[1]` has the largest sum 1.

## Example 3:
## *Ví dụ 3:*

**Input:** `nums = [5,4,-1,7,8]`
**Output:** `23`
**Explanation:** The subarray `[5,4,-1,7,8]` has the largest sum 23.

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 10^5`
*   `-10^4 <= nums[i] <= 10^4`
