# Result for Univalued Binary Tree
# *Kết quả cho bài toán Cây Nhị phân Đơn trị*

## Description
## *Mô tả*

A binary tree is **univalued** if every node in the tree has the same value.
*Một cây nhị phân được gọi là **đơn trị** nếu mọi nút trong cây đều có cùng một giá trị.*

Given the `root` of a binary tree, return `true` if the given tree is **univalued**, or `false` otherwise.
*Cho `gốc` của một cây nhị phân, hãy trả về `true` nếu cây đã cho là **đơn trị**, hoặc `false` nếu ngược lại.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** root = [1,1,1,1,1,null,1]
**Output:** true

## Example 2:
## *Ví dụ 2:*

**Input:** root = [2,2,2,5,2]
**Output:** false

---

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the tree is in the range `[1, 100]`.
*   `0 <= Node.val <= 99`
