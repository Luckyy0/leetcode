# Result for Power of Three
# *Kết quả cho bài toán Lũy thừa của Ba*

## Description
## *Mô tả*

Given an integer `n`, return `true` *if it is a power of three. Otherwise, return* `false`.
*Cho một số nguyên `n`, trả về `true` *nếu nó là lũy thừa của ba. Ngược lại, trả về* `false`.*

An integer `n` is a power of three, if there exists an integer `x` such that `n == 3^x`.
*Một số nguyên `n` là lũy thừa của ba, nếu tồn tại một số nguyên `x` sao cho `n == 3^x`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 27`
**Output:** `true`

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 0`
**Output:** `false`

## Constraints:
## *Ràng buộc:*

*   `-2^31 <= n <= 2^31 - 1`

**Follow up:** Could you do it without using any loop / recursion?
**Câu hỏi mở rộng:** Bạn có thể thực hiện nó mà không sử dụng bất kỳ vòng lặp / đệ quy nào không?
