# Result for Ugly Number
# *Kết quả cho bài toán Số xấu*

## Description
## *Mô tả*

An **ugly number** is a positive integer whose prime factors are limited to `2`, `3`, and `5`.
*Một **số xấu** là một số nguyên dương mà các thừa số nguyên tố của nó chỉ giới hạn trong `2`, `3`, và `5`.*

Given an integer `n`, return `true` *if* `n` *is an **ugly number***.
*Cho một số nguyên `n`, trả về `true` *nếu* `n` *là một **số xấu***.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 6`
**Output:** `true`
**Explanation:** 6 = 2 × 3

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 1`
**Output:** `true`
**Explanation:** 1 has no prime factors, therefore all of its prime factors are limited to 2, 3, and 5.

## Example 3:
## *Ví dụ 3:*

**Input:** `n = 14`
**Output:** `false`
**Explanation:** 14 is not ugly since it includes the prime factor 7.

## Constraints:
## *Ràng buộc:*

*   `-2^31 <= n <= 2^31 - 1`
