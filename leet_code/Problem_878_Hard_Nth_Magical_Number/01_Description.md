# Result for Nth Magical Number
# *Kết quả cho bài toán Số Kỳ ảo thứ N*

## Description
## *Mô tả*

A positive integer is **magical** if it is divisible by either `a` or `b`.
*Một số nguyên dương được gọi là **kỳ ảo** nếu nó chia hết cho `a` hoặc `b`.*

Given the three integers `n`, `a`, and `b`, return *the `n`-th magical number*. Since the answer may be very large, **return it modulo `10^9 + 7`**.
*Cho ba số nguyên `n`, `a` và `b`, hãy trả về *số kỳ ảo thứ `n`*. Vì câu trả lời có thể rất lớn, hãy **trả về kết quả theo modulo `10^9 + 7`**.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** n = 1, a = 2, b = 3
**Output:** 2

## Example 2:
## *Ví dụ 2:*

**Input:** n = 4, a = 2, b = 3
**Output:** 6

---

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 10^9`
*   `2 <= a, b <= 40000`
