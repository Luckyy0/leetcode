# Result for Maximum Depth of Binary Tree
# *Kết quả cho bài toán Chiều Sâu Tối Đa của Cây Nhị Phân*

## Description
## *Mô tả*

Given the `root` of a binary tree, return *its maximum depth*.
*Cho `root` của một cây nhị phân, hãy trả về *chiều sâu tối đa của nó***.*

A binary tree's **maximum depth** is the number of nodes along the longest path from the root node down to the farthest leaf node.
*Chiều sâu tối đa** của một cây nhị phân là số lượng nút dọc theo đường đi dài nhất từ nút gốc xuống nút lá xa nhất.*

## Example 1:
## *Ví dụ 1:*

**Input:** `root = [3,9,20,null,null,15,7]`
**Output:** `3`

## Example 2:
## *Ví dụ 2:*

**Input:** `root = [1,null,2]`
**Output:** `2`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the tree is in the range `[0, 10^4]`.
*   `-100 <= Node.val <= 100`
