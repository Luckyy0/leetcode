# Result for Single Element in a Sorted Array
# *Kết quả cho bài toán Phần tử Đơn lẻ trong Mảng Đã sắp xếp*

## Description
## *Mô tả*

You are given a sorted array consisting of only integers where every element appears exactly twice, except for one element which appears exactly once.
*Bạn được cho một mảng đã sắp xếp chỉ bao gồm các số nguyên, trong đó mỗi phần tử xuất hiện chính xác hai lần, ngoại trừ một phần tử xuất hiện chính xác một lần.*

Return *the single element that appears only once*.
*Trả về *phần tử đơn lẻ xuất hiện chỉ một lần*.*

Your solution must run in `O(log n)` time and `O(1)` space.
*Giải pháp của bạn phải chạy trong thời gian `O(log n)` và không gian `O(1)`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [1,1,2,3,3,4,4,8,8]`
**Output:** `2`

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [3,3,7,7,10,11,11]`
**Output:** `10`

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 10^5`
*   `0 <= nums[i] <= 10^5`
