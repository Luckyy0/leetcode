# Result for Same Tree
# *Kết quả cho bài toán Cây Giống Nhau*

## Description
## *Mô tả*

Given the roots of two binary trees `p` and `q`, write a function to check if they are the same or not.
*Cho gốc của hai cây nhị phân `p` và `q`, hãy viết một hàm để kiểm tra xem chúng có giống nhau hay không.*

Two binary trees are considered the same if they are structurally identical, and the nodes have the same value.
*Hai cây nhị phân được coi là giống nhau nếu chúng giống hệt nhau về cấu trúc và các nút có cùng giá trị.*

## Example 1:
## *Ví dụ 1:*

**Input:** `p = [1,2,3], q = [1,2,3]`
**Output:** `true`

## Example 2:
## *Ví dụ 2:*

**Input:** `p = [1,2], q = [1,null,2]`
**Output:** `false`

## Example 3:
## *Ví dụ 3:*

**Input:** `p = [1,2,1], q = [1,1,2]`
**Output:** `false`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in both trees is in the range `[0, 100]`.
*   `-10^4 <= Node.val <= 10^4`
