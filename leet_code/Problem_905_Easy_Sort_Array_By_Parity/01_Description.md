# Result for Sort Array By Parity
# *Kết quả cho bài toán Sắp xếp Mảng Theo Tính Chẵn Lẻ*

## Description
## *Mô tả*

Given an integer array `nums`, move all the even integers at the beginning of the array followed by all the odd integers.
*Cho một mảng số nguyên `nums`, hãy di chuyển tất cả các số nguyên chẵn lên đầu mảng, sau đó là tất cả các số nguyên lẻ.*

Return *any array that satisfies this condition*.
*Hãy trả về *bất kỳ mảng nào thỏa mãn điều kiện này*.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** nums = [3,1,2,4]
**Output:** [2,4,3,1]
**Explanation:** The outputs [4,2,3,1], [2,4,1,3], and [4,2,1,3] would also be accepted.

## Example 2:
## *Ví dụ 2:*

**Input:** nums = [0]
**Output:** [0]

---

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 5000`
*   `0 <= nums[i] <= 5000`
