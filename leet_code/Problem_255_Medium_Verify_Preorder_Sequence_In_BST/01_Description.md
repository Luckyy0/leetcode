# Result for Verify Preorder Sequence in Binary Search Tree
# *Kết quả cho bài toán Xác minh chuỗi Preorder trong Cây tìm kiếm nhị phân*

## Description
## *Mô tả*

Given an array of **unique** integers `preorder`, return `true` *if it is the correct preorder traversal sequence of a binary search tree*.
*Cho một mảng các số nguyên **duy nhất** `preorder`, trả về `true` *nếu nó là chuỗi duyệt tiền thứ tự chính xác của một cây tìm kiếm nhị phân*.*

## Example 1:
## *Ví dụ 1:*

**Input:** `preorder = [5,2,1,3,6]`
**Output:** `true`

## Example 2:
## *Ví dụ 2:*

**Input:** `preorder = [5,2,6,1,3]`
**Output:** `false`

## Constraints:
## *Ràng buộc:*

*   `1 <= preorder.length <= 10^4`
*   `1 <= preorder[i] <= 10^4`
*   All the elements of `preorder` are **unique**.

**Follow up:** Could you do it using only constant space complexity?
**Câu hỏi mở rộng:** Bạn có thể làm điều đó chỉ với độ phức tạp không gian không đổi không?
