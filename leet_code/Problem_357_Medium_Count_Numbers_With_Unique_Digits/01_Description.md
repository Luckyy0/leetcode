# Result for Count Numbers with Unique Digits
# *Kết quả cho bài toán Đếm Số có Chữ số Duy nhất*

## Description
## *Mô tả*

Given an integer `n`, return the count of all numbers with unique digits, `x`, where `0 <= x < 10^n`.
*Cho một số nguyên `n`, hãy trả về số lượng tất cả các số có các chữ số duy nhất, `x`, trong đó `0 <= x < 10^n`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 2`
**Output:** `91`
**Explanation:** The answer should be the total numbers in the range of 0 <= x < 100, excluding 11, 22, 33, 44, 55, 66, 77, 88, 99.

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 0`
**Output:** `1`

## Constraints:
## *Ràng buộc:*

*   `0 <= n <= 8`
