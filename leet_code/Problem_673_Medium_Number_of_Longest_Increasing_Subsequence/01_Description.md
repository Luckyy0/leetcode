# Result for Number of Longest Increasing Subsequence
# *Kết quả cho bài toán Số lượng Dãy con Tăng dài nhất*

## Description
## *Mô tả*

Given an integer array `nums`, return the number of longest increasing subsequences.
*Cho một mảng số nguyên `nums`, hãy trả về số lượng các dãy con tăng dài nhất.*

Notice that the sequence has to be **strictly** increasing.
*Lưu ý rằng dãy phải là tăng **nghiêm ngặt**.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** nums = [1,3,5,4,7]
**Output:** 2
**Explanation:** The two longest increasing subsequences are [1, 3, 4, 7] and [1, 3, 5, 7].

## Example 2:
## *Ví dụ 2:*

**Input:** nums = [2,2,2,2,2]
**Output:** 5
**Explanation:** The length of the longest increasing subsequence is 1, and there are 5 subsequences of length 1, so output 5.

---

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 2000`
*   `-10^6 <= nums[i] <= 10^6`
