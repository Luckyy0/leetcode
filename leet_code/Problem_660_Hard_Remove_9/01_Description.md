# Result for Remove 9
# *Kết quả cho bài toán Loại bỏ số 9*

## Description
## *Mô tả*

Start from integer `1`, remove any integer that contains `9` such as `9`, `19`, `29`, `90`, `91`, ...
*Bắt đầu từ số nguyên `1`, hãy loại bỏ bất kỳ số nguyên nào có chứa chữ số `9`, ví dụ như `9`, `19`, `29`, `90`, `91`, ...*

Now, you will have a new integer sequence `1, 2, 3, 4, 5, 6, 7, 8, 10, 11, ...`
*Bây giờ, bạn sẽ có một chuỗi số nguyên mới: `1, 2, 3, 4, 5, 6, 7, 8, 10, 11, ...`*

Given an integer `n`, return the `n-th` element of this new sequence.
*Cho một số nguyên `n`, hãy trả về phần tử thứ `n` của chuỗi mới này.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** n = 9
**Output:** 10

## Example 2:
## *Ví dụ 2:*

**Input:** n = 10
**Output:** 11

---

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 8 * 10^8`
