# Result for Maximum Size Subarray Sum Equals k
# *Kết quả cho bài toán Kích thước Lớn nhất của Mảng con có Tổng bằng k*

## Description
## *Mô tả*

Given an integer array `nums` and an integer `k`, return *the maximum length of a subarray that sums to* `k`. If there is not one, return `0` instead.
*Cho một mảng số nguyên `nums` và một số nguyên `k`, trả về *độ dài lớn nhất của một mảng con có tổng bằng* `k`. Nếu không có mảng con nào như vậy, trả về `0` thay thế.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [1,-1,5,-2,3], k = 3`
**Output:** `4`
**Explanation:** The subarray [1, -1, 5, -2] sums to 3 and is the longest.

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [-2,-1,2,1], k = 1`
**Output:** `2`
**Explanation:** The subarray [-1, 2] sums to 1 and is the longest.

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 2 * 10^5`
*   `-10^4 <= nums[i] <= 10^4`
*   `-10^9 <= k <= 10^9`
