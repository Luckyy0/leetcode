# Result for Longest Substring Without Repeating Characters
# *Kết quả cho bài toán Chuỗi Con Dài Nhất Không Lặp Lại Ký Tự*

## Description
## *Mô tả*

Given a string `s`, find the length of the **longest substring** without repeating characters.
*Cho một chuỗi `s`, hãy tìm độ dài của **chuỗi con dài nhất** không có ký tự lặp lại.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = "abcabcbb"`
**Output:** `3`
**Explanation:** The answer is "abc", with the length of 3.
***Giải thích:** Câu trả lời là "abc", với độ dài là 3.*

## Example 2:
## *Ví dụ 2:*

**Input:** `s = "bbbbb"`
**Output:** `1`
**Explanation:** The answer is "b", with the length of 1.
***Giải thích:** Câu trả lời là "b", với độ dài là 1.*

## Example 3:
## *Ví dụ 3:*

**Input:** `s = "pwwkew"`
**Output:** `3`
**Explanation:** The answer is "wke", with the length of 3.
Notice that the answer must be a substring, "pwke" is a subsequence and not a substring.
***Giải thích:** Câu trả lời là "wke", với độ dài là 3.
Lưu ý rằng câu trả lời phải là một chuỗi con, "pwke" là một dãy con và không phải là chuỗi con.*

## Constraints:
## *Ràng buộc:*

*   `0 <= s.length <= 5 * 10^4`
*   `s` consists of English letters, digits, symbols and spaces.
*   *`s` bao gồm các chữ cái tiếng Anh, chữ số, ký hiệu và khoảng trắng.*
