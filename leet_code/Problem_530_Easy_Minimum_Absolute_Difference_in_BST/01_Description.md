# Result for Minimum Absolute Difference in BST
# *Kết quả cho bài toán Hiệu Tuyệt đối Nhỏ nhất trong BST*

## Description
## *Mô tả*

Given the `root` of a Binary Search Tree (BST), return *the minimum absolute difference between the values of any two different nodes in the tree*.
*Cho `root` của một Cây Nhị phân Tìm kiếm (BST), hãy trả về *hiệu tuyệt đối nhỏ nhất giữa các giá trị của bất kỳ hai nút khác nhau nào trong cây*.*

## Example 1:
## *Ví dụ 1:*

**Input:** `root = [4,2,6,1,3]`
**Output:** `1`

## Example 2:
## *Ví dụ 2:*

**Input:** `root = [1,0,48,null,null,12,49]`
**Output:** `1`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the tree is in the range `[2, 10^4]`.
*   `0 <= Node.val <= 10^5`
