# Result for Minimum Factorization
# *Kết quả cho bài toán Phân tích nhân tử Tối thiểu*

## Description
## *Mô tả*

Given a positive integer `a`, find the smallest positive integer `b` whose multiplication of each digit equals to `a`.
*Cho một số nguyên dương `a`, hãy tìm số nguyên dương nhỏ nhất `b` sao cho tích của các chữ số của nó bằng `a`.*

If there is no such answer or the answer is not fit in 32-bit signed integer, then return `0`.
*Nếu không có câu trả lời nào như vậy hoặc câu trả lời không vừa với số nguyên có dấu 32-bit, thì trả về `0`.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** a = 48
**Output:** 68

## Example 2:
## *Ví dụ 2:*

**Input:** a = 15
**Output:** 35

---

## Constraints:
## *Ràng buộc:*

*   `1 <= a <= 10^9`
