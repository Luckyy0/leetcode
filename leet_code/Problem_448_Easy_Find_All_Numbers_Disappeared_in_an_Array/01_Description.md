# Result for Find All Numbers Disappeared in an Array
# *Kết quả cho bài toán Tìm tất cả các Số bị mất trong Mảng*

## Description
## *Mô tả*

Given an array `nums` of `n` integers where `nums[i]` is in the range `[1, n]`, return *an array of all the integers in the range `[1, n]` that do not appear in `nums`*.
*Cho một mảng `nums` gồm `n` số nguyên trong đó `nums[i]` nằm trong khoảng `[1, n]`, hãy trả về *một mảng chứa tất cả các số nguyên trong khoảng `[1, n]` không xuất hiện trong `nums`***.*

You must write an algorithm that runs in $O(n)$ time and uses only $O(1)$ extra space.
*Bạn phải viết một thuật toán chạy trong thời gian $O(n)$ và chỉ sử dụng $O(1)$ không gian bộ nhớ bổ sung.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [4,3,2,7,8,2,3,1]`
**Output:** `[5,6]`

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [1,1]`
**Output:** `[2]`

## Constraints:
## *Ràng buộc:*

*   `n == nums.length`
*   `1 <= n <= 10^5`
*   `1 <= nums[i] <= n`
