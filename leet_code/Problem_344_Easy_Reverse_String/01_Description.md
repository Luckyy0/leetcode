# Result for Reverse String
# *Kết quả cho bài toán Đảo ngược Chuỗi*

## Description
## *Mô tả*

Write a function that reverses a string. The input string is given as an array of characters `s`.
*Viết một hàm đảo ngược một chuỗi. Chuỗi đầu vào được cho dưới dạng một mảng các ký tự `s`.*

You must do this by modifying the input array **in-place** with `O(1)` extra memory.
*Bạn phải thực hiện điều này bằng cách sửa đổi mảng đầu vào **tại chỗ** với bộ nhớ phụ `O(1)`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = ["h","e","l","l","o"]`
**Output:** `["o","l","l","e","h"]`

## Example 2:
## *Ví dụ 2:*

**Input:** `s = ["H","a","n","n","a","h"]`
**Output:** `["h","a","n","n","a","H"]`

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 10^5`
*   `s[i]` is a printable ascii character.
