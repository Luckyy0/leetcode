# Result for Unique Binary Search Trees
# *Kết quả cho bài toán Cây Tìm Kiếm Nhị Phân Duy Nhất*

## Description
## *Mô tả*

Given an integer `n`, return *the number of structurally unique **BST's** (binary search trees) which has exactly `n` nodes of unique values from `1` to `n`*.
*Cho một số nguyên `n`, hãy trả về *số lượng các **BST** (cây tìm kiếm nhị phân) có cấu trúc duy nhất có chính xác `n` nút với các giá trị duy nhất từ `1` đến `n`**.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 3`
**Output:** `5`

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 1`
**Output:** `1`

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 19`
