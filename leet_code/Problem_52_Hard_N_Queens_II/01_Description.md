# Result for N-Queens II
# *Kết quả cho bài toán N-Queens II*

## Description
## *Mô tả*

The **n-queens** puzzle is the problem of placing `n` queens on an `n x n` chessboard such that no two queens attack each other.
*Bài toán **n-queens** là bài toán đặt `n` quân hậu trên một bàn cờ `n x n` sao cho không có hai quân hậu nào tấn công nhau.*

Given an integer `n`, return *the number of distinct solutions to the **n-queens puzzle***.
*Cho một số nguyên `n`, hãy trả về *số lượng các giải pháp riêng biệt cho **bài toán n-queens***.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 4`
**Output:** `2`
**Explanation:** There are two distinct solutions to the 4-queens puzzle as shown.

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 1`
**Output:** `1`

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 9`
