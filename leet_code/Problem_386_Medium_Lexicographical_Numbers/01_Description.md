# Result for Lexicographical Numbers
# *Kết quả cho bài toán Số theo Thứ tự Từ điển*

## Description
## *Mô tả*

Given an integer `n`, return all the integers in the range `[1, n]` sorted in lexicographical order.
*Cho một số nguyên `n`, hãy trả về tất cả các số nguyên trong phạm vi `[1, n]` được sắp xếp theo thứ tự từ điển.*

You must write an algorithm that runs in `O(n)` time and uses `O(1)` extra space.
*Bạn phải viết một thuật toán chạy trong thời gian `O(n)` và sử dụng không gian phụ `O(1)`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 13`
**Output:** `[1,10,11,12,13,2,3,4,5,6,7,8,9]`

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 2`
**Output:** `[1,2]`

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 5 * 10^4`
