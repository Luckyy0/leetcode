# Result for Binary Number with Alternating Bits
# *Kết quả cho bài toán Số Nhị phân có các Bit xen kẽ*

## Description
## *Mô tả*

Given a positive integer, check whether it has alternating bits: namely, if two adjacent bits will always have different values.
*Cho một số nguyên dương, hãy kiểm tra xem nó có các bit xen kẽ hay không: có nghĩa là hai bit liền kề sẽ luôn có các giá trị khác nhau.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** n = 5
**Output:** true
**Explanation:** The binary representation of 5 is: 101

## Example 2:
## *Ví dụ 2:*

**Input:** n = 7
**Output:** false
**Explanation:** The binary representation of 7 is: 111.

## Example 3:
## *Ví dụ 3:*

**Input:** n = 11
**Output:** false
**Explanation:** The binary representation of 11 is: 1011.

---

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 2^31 - 1`
