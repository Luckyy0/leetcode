# Result for Letter Case Permutation
# *Kết quả cho bài toán Hoán vị Chữ hoa Chữ thường*

## Description
## *Mô tả*

Given a string `s`, you can transform every letter individually to be lowercase or uppercase to create another string.
*Cho một chuỗi `s`, bạn có thể chuyển đổi từng chữ cái riêng lẻ thành chữ thường hoặc chữ hoa để tạo ra một chuỗi khác.*

Return *a list of all possible strings we could create*.
*Trả về *một danh sách tất cả các chuỗi có thể mà chúng ta có thể tạo ra*.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** s = "a1b2"
**Output:** ["a1b2","a1B2","A1b2","A1B2"]

## Example 2:
## *Ví dụ 2:*

**Input:** s = "3z4"
**Output:** ["3z4","3Z4"]

---

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 12`
*   `s` consists of lowercase English letters, uppercase English letters, and digits.
