# Result for Number of Digit One
# *Kết quả cho bài toán Số lượng chữ số một*

## Description
## *Mô tả*

Given an integer `n`, count *the total number of digit `1` appearing in all non-negative integers less than or equal to `n`*.
*Cho một số nguyên `n`, đếm *tổng số chữ số `1` xuất hiện trong tất cả các số nguyên không âm nhỏ hơn hoặc bằng `n`*.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 13`
**Output:** `6`
**Explanation:** The digit 1 appears in the following numbers: 1, 10, 11, 12, 13.
Note that 11 contains two 1s. Total count is 1 + 1 + 2 + 1 + 1 = 6.

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 0`
**Output:** `0`

## Constraints:
## *Ràng buộc:*

*   `0 <= n <= 10^9`
