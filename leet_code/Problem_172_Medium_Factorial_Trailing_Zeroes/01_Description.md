# Result for Factorial Trailing Zeroes
# *Kết quả cho bài toán Số chữ số 0 tận cùng của giai thừa*

## Description
## *Mô tả*

Given an integer `n`, return *the number of trailing zeroes in `n!`*.
*Cho một số nguyên `n`, hãy trả về *số lượng chữ số 0 tận cùng của `n!`***.*

Note that `n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1`.
*Lưu ý rằng `n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 3`
**Output:** `0`
**Explanation:** 3! = 6, no trailing zero.

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 5`
**Output:** `1`
**Explanation:** 5! = 120, one trailing zero.

## Example 3:
## *Ví dụ 3:*

**Input:** `n = 0`
**Output:** `0`

## Constraints:
## *Ràng buộc:*

*   `0 <= n <= 10^4`

**Follow up:** Could you write an algorithm that runs in **O(log n)** time complexity?
**Câu hỏi mở rộng:** Bạn có thể viết một thuật toán chạy trong độ phức tạp thời gian **O(log n)** không?
