# Result for N-ary Tree Level Order Traversal
# *Kết quả cho bài toán Duyệt cây N-phân theo Mức*

## Description
## *Mô tả*

Given an n-ary tree, return *the level order traversal of its nodes' values*.
*Cho một cây n-phân, hãy trả về *kết quả duyệt các giá trị của các nút theo mức (level order)***.*

Nary-Tree input serialization is represented in their level order traversal, each group of children is separated by the null value (See examples).
*Dữ liệu đầu vào của cây N-phân được biểu diễn theo thứ tự duyệt theo mức, mỗi nhóm con được phân tách bằng giá trị null (Xem ví dụ).*

## Example 1:
## *Ví dụ 1:*

**Input:** `root = [1,null,3,2,4,null,5,6]`
**Output:** `[[1],[3,2,4],[5,6]]`

## Constraints:
## *Ràng buộc:*

*   The total number of nodes is between `0` and `10^4`.
