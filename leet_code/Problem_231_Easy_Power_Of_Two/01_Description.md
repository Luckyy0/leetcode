# Result for Power of Two
# *Kết quả cho bài toán Lũy thừa của 2*

## Description
## *Mô tả*

Given an integer `n`, return `true` if it is a power of two. Otherwise, return `false`.
*Cho một số nguyên `n`, trả về `true` nếu nó là lũy thừa của 2. Ngược lại, trả về `false`.*

An integer `n` is a power of two, if there exists an integer `x` such that `n == 2^x`.
*Một số nguyên `n` là lũy thừa của 2, nếu tồn tại một số nguyên `x` sao cho `n == 2^x`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 1`
**Output:** `true`
**Explanation:** 2^0 = 1

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 16`
**Output:** `true`
**Explanation:** 2^4 = 16

## Example 3:
## *Ví dụ 3:*

**Input:** `n = 3`
**Output:** `false`

## Constraints:
## *Ràng buộc:*

*   `-2^31 <= n <= 2^31 - 1`

**Câu hỏi mở rộng:** Bạn có thể giải quyết mà không cần vòng lặp/đệ quy không?
