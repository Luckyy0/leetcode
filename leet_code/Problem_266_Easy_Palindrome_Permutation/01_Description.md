# Result for Palindrome Permutation
# *Kết quả cho bài toán Hoán vị Palindrome*

## Description
## *Mô tả*

Given a string `s`, return `true` *if a permutation of the string could form a **palindrome** and* `false` *otherwise*.
*Cho một chuỗi `s`, hãy trả về `true` *nếu một hoán vị của chuỗi có thể tạo thành một **palindrome** và* `false` *nếu không*.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = "code"`
**Output:** `false`

## Example 2:
## *Ví dụ 2:*

**Input:** `s = "aab"`
**Output:** `true`

## Example 3:
## *Ví dụ 3:*

**Input:** `s = "carerac"`
**Output:** `true`

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 5000`
*   `s` consists of only lowercase English letters.
