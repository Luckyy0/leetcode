# Result for Reordered Power of 2
# *Kết quả cho bài toán Sắp xếp lại thành Lũy thừa của 2*

## Description
## *Mô tả*

You are given an integer `n`. We reorder the digits in any order (including the original order) such that the leading digit is not zero.
*Bạn được cho một số nguyên `n`. Chúng ta sắp xếp lại các chữ số theo bất kỳ thứ tự nào (bao gồm cả thứ tự ban đầu) sao cho chữ số dẫn đầu không phải là số không.*

Return `true` *if and only if we can do this so that the resulting number is a power of two*.
*Trả về `true` *nếu và chỉ nếu chúng ta có thể làm điều này để số kết quả là một lũy thừa của hai*.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** n = 1
**Output:** true

## Example 2:
## *Ví dụ 2:*

**Input:** n = 10
**Output:** false

---

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 10^9`
