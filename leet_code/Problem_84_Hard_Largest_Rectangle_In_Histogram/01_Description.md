# Result for Largest Rectangle in Histogram
# *Kết quả cho bài toán Hình Chữ Nhật Lớn Nhất Trong Biểu Đồ Cột*

## Description
## *Mô tả*

Given an array of integers `heights` representing the histogram's bar height where the width of each bar is `1`, return *the area of the largest rectangle in the histogram*.
*Cho một mảng các số nguyên `heights` đại diện cho chiều cao các cột của biểu đồ, trong đó chiều rộng của mỗi cột là `1`, hãy trả về *diện tích của hình chữ nhật lớn nhất trong biểu đồ đó**.*

## Example 1:
## *Ví dụ 1:*

**Input:** `heights = [2,1,5,6,2,3]`
**Output:** `10`
**Explanation:** The above is a histogram where width of each bar is 1.
The largest rectangle is shown in the shaded area, which has an area = 10 units.

## Example 2:
## *Ví dụ 2:*

**Input:** `heights = [2,4]`
**Output:** `4`

## Constraints:
## *Ràng buộc:*

*   `1 <= heights.length <= 10^5`
*   `0 <= heights[i] <= 10^4`
