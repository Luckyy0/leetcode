# Result for Diagonal Traverse
# *Kết quả cho bài toán Duyệt Chéo*

## Description
## *Mô tả*

Given an `m x n` matrix `mat`, return *an array of all the elements of the array in a diagonal order*.
*Cho một ma trận `m x n` `mat`, hãy trả về *một mảng chứa tất cả các phần tử của mảng theo thứ tự đường chéo*.*

## Example 1:
## *Ví dụ 1:*

**Input:** `mat = [[1,2,3],[4,5,6],[7,8,9]]`
**Output:** `[1,2,4,7,5,3,6,8,9]`
**Explanation:**
1 -> 2 -> 4 -> 7 -> 5 -> 3 -> 6 -> 8 -> 9

## Example 2:
## *Ví dụ 2:*

**Input:** `mat = [[1,2],[3,4]]`
**Output:** `[1,2,3,4]`

## Constraints:
## *Ràng buộc:*

*   `m == mat.length`
*   `n == mat[i].length`
*   `1 <= m, n <= 10^4`
*   `1 <= m * n <= 10^4`
*   `-10^5 <= mat[i][j] <= 10^5`
