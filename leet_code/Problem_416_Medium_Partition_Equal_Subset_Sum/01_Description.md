# Result for Partition Equal Subset Sum
# *Kết quả cho bài toán Chia mảng thành hai tập hợp có Tổng bằng nhau*

## Description
## *Mô tả*

Given an integer array `nums`, return `true` if you can partition the array into two subsets such that the sum of the elements in both subsets is equal or `false` otherwise.
*Cho một mảng số nguyên `nums`, hãy trả về `true` nếu bạn có thể chia mảng thành hai tập hợp con sao cho tổng các phần tử trong cả hai tập hợp con là bằng nhau, ngược lại trả về `false`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [1,5,11,5]`
**Output:** `true`
**Explanation:** The array can be partitioned as [1, 5, 5] and [11].

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [1,2,3,5]`
**Output:** `false`
**Explanation:** The array cannot be partitioned into equal sum subsets.

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 200`
*   `1 <= nums[i] <= 100`
