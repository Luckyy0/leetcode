# Result for Strobogrammatic Number II
# *Kết quả cho bài toán Số Strobogrammatic II*

## Description
## *Mô tả*

Given an integer `n`, return all the **strobogrammatic numbers** that are of length `n`. You may return the answer in **any order**.
*Cho một số nguyên `n`, hãy trả về tất cả **các số strobogrammatic** có độ dài `n`. Bạn có thể trả về câu trả lời theo **bất kỳ thứ tự nào**.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 2`
**Output:** `["11","69","88","96"]`

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 1`
**Output:** `["0","1","8"]`

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 14`
