# Result for Find Largest Value in Each Tree Row
# *Kết quả cho bài toán Tìm Giá trị Lớn nhất trong Mỗi Hàng Cây*

## Description
## *Mô tả*

Given the `root` of a binary tree, return *an array of the largest value in each row* (0-indexed).
*Cho `root` của một cây nhị phân, hãy trả về *một mảng chứa giá trị lớn nhất trong mỗi hàng* (chỉ số bắt đầu từ 0).*

## Example 1:
## *Ví dụ 1:*

**Input:** `root = [1,3,2,5,3,null,9]`
**Output:** `[1,3,9]`

## Example 2:
## *Ví dụ 2:*

**Input:** `root = [1,2,3]`
**Output:** `[1,3]`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the tree will be in the range `[0, 10^4]`.
*   `-2^{31} <= Node.val <= 2^{31} - 1`
