# Result for Remove Duplicates from Sorted List
# *Kết quả cho bài toán Loại Bỏ Phần Tử Trùng Lặp Khỏi Danh Sách Đã Sắp Xếp*

## Description
## *Mô tả*

Given the `head` of a sorted linked list, delete all duplicates such that each element appears only once. Return the linked list **sorted** as well.
*Cho `head` của một danh sách liên kết đã được sắp xếp, hãy xóa tất cả các phần tử trùng lặp sao cho mỗi phần tử chỉ xuất hiện một lần. Trả về danh sách liên kết cũng đã được **sắp xếp**.*

## Example 1:
## *Ví dụ 1:*

**Input:** `head = [1,1,2]`
**Output:** `[1,2]`

## Example 2:
## *Ví dụ 2:*

**Input:** `head = [1,1,2,3,3]`
**Output:** `[1,2,3]`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the list is in the range `[0, 300]`.
*   `-100 <= Node.val <= 100`
*   The list is guaranteed to be **sorted** in ascending order.
