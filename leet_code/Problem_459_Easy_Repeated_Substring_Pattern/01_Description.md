# Result for Repeated Substring Pattern
# *Kết quả cho bài toán Mẫu Chuỗi con được Lặp lại*

## Description
## *Mô tả*

Given a string `s`, check if it can be constructed by taking a substring of it and appending multiple copies of the substring together.
*Cho một chuỗi `s`, hãy kiểm tra xem nó có thể được tạo ra bằng cách lấy một chuỗi con của nó và ghép nhiều bản sao của chuỗi con đó lại với nhau hay không.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = "abab"`
**Output:** `true`
**Explanation:** It is the substring "ab" twice.

## Example 2:
## *Ví dụ 2:*

**Input:** `s = "aba"`
**Output:** `false`

## Example 3:
## *Ví dụ 3:*

**Input:** `s = "abcabcabcabc"`
**Output:** `true`
**Explanation:** It is the substring "abc" four times or the substring "abcabc" twice.

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 10^4`
*   `s` consists of lowercase English letters.
