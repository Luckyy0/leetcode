# Result for Binary Tree Level Order Traversal II
# *Kết quả cho bài toán Duyệt Cây Nhị Phân Theo Tầng II*

## Description
## *Mô tả*

Given the `root` of a binary tree, return *the bottom-up level order traversal of its nodes' values*. (i.e., from left to right, level by level from leaf to root).
*Cho `root` của một cây nhị phân, hãy trả về *phép duyệt theo thứ tự tầng từ dưới lên của các giá trị nút*. (tức là, từ trái sang phải, từng tầng một từ lá lên đến gốc).*

## Example 1:
## *Ví dụ 1:*

**Input:** `root = [3,9,20,null,null,15,7]`
**Output:** `[[15,7],[9,20],[3]]`

## Example 2:
## *Ví dụ 2:*

**Input:** `root = [1]`
**Output:** `[[1]]`

## Example 3:
## *Ví dụ 3:*

**Input:** `root = []`
**Output:** `[]`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the tree is in the range `[0, 2000]`.
*   `-1000 <= Node.val <= 1000`
