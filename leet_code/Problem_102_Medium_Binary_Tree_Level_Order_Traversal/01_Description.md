# Result for Binary Tree Level Order Traversal
# *Kết quả cho bài toán Duyệt Cây Nhị Phân Theo Tầng*

## Description
## *Mô tả*

Given the `root` of a binary tree, return *the level order traversal of its nodes' values*. (i.e., from left to right, level by level).
*Cho `root` của một cây nhị phân, hãy trả về *phép duyệt theo thứ tự tầng của các giá trị nút*. (tức là, từ trái sang phải, từng tầng một).*

## Example 1:
## *Ví dụ 1:*

**Input:** `root = [3,9,20,null,null,15,7]`
**Output:** `[[3],[9,20],[15,7]]`

## Example 2:
## *Ví dụ 2:*

**Input:** `root = [1]`
**Output:** `[[1]]`

## Example 3:
## *Ví dụ 3:*

**Input:** `root = []`
**Output:** `[]`

## Constraints:
## *Ràng buộc:*

*   The number of nodes in the tree is in the range `[0, 2000]`.
*   `-1000 <= Node.val <= 1000`
