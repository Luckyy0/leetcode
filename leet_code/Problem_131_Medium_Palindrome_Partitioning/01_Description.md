# Result for Palindrome Partitioning
# *Kết quả cho bài toán Phân Tách Chuỗi Đối Xứng*

## Description
## *Mô tả*

Given a string `s`, partition `s` such that every substring of the partition is a **palindrome**. Return *all possible palindrome partitioning of `s`*.
*Cho một chuỗi `s`, hãy phân tách `s` sao cho mọi chuỗi con của phép phân tách đều là một **chuỗi đối xứng**. Hãy trả về *tất cả các phép phân tách chuỗi đối xứng có thể có của `s`**.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = "aab"`
**Output:** `[["a","a","b"],["aa","b"]]`

## Example 2:
## *Ví dụ 2:*

**Input:** `s = "a"`
**Output:** `[["a"]]`

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 16`
*   `s` contains only lowercase English letters.
