# Result for Generate Parentheses
# *Kết quả cho bài toán Tạo Dấu Ngoặc*

## Description
## *Mô tả*

Given `n` pairs of parentheses, write a function to generate all combinations of well-formed parentheses.
*Cho `n` cặp dấu ngoặc, hãy viết một hàm để tạo tất cả các tổ hợp dấu ngoặc hợp lệ.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 3`
**Output:** `["((()))","(()())","(())()","()(())","()()()"]`

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 1`
**Output:** `["()"]`

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 8`
