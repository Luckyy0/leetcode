# Result for Fibonacci Number
# *Kết quả cho bài toán Số Fibonacci*

## Description
## *Mô tả*

The **Fibonacci numbers**, commonly denoted `F(n)` form a sequence, called the **Fibonacci sequence**, such that each number is the sum of the two preceding ones, starting from `0` and `1`. That is,
***Các số Fibonacci**, thường được ký hiệu là `F(n)` tạo thành một dãy, được gọi là **dãy Fibonacci**, sao cho mỗi số là tổng của hai số liền trước, bắt đầu từ `0` và `1`. Tức là,*

`F(0) = 0`, `F(1) = 1`
`F(n) = F(n - 1) + F(n - 2)`, for `n > 1`.

Given `n`, calculate `F(n)`.
*Cho `n`, hãy tính `F(n)`.*

## Example 1:
## *Ví dụ 1:*

**Input:** `n = 2`
**Output:** `1`
**Explanation:** F(2) = F(1) + F(0) = 1 + 0 = 1.

## Example 2:
## *Ví dụ 2:*

**Input:** `n = 3`
**Output:** `2`
**Explanation:** F(3) = F(2) + F(1) = 1 + 1 = 2.

## Example 3:
## *Ví dụ 3:*

**Input:** `n = 4`
**Output:** `3`
**Explanation:** F(4) = F(3) + F(2) = 2 + 1 = 3.

## Constraints:
## *Ràng buộc:*

*   `0 <= n <= 30`
