# Result for Contains Duplicate
# *Kết quả cho bài toán Chứa bản sao*

## Description
## *Mô tả*

Given an integer array `nums`, return `true` if any value appears **at least twice** in the array, and return `false` if every element is distinct.
*Cho một mảng số nguyên `nums`, trả về `true` nếu bất kỳ giá trị nào xuất hiện **ít nhất hai lần** trong mảng, và trả về `false` nếu mọi phần tử là riêng biệt.*

## Example 1:
## *Ví dụ 1:*

**Input:** `nums = [1,2,3,1]`
**Output:** `true`

## Example 2:
## *Ví dụ 2:*

**Input:** `nums = [1,2,3,4]`
**Output:** `false`

## Example 3:
## *Ví dụ 3:*

**Input:** `nums = [1,1,1,3,3,4,3,2,4,2]`
**Output:** `true`

## Constraints:
## *Ràng buộc:*

*   `1 <= nums.length <= 10^5`
*   `-10^9 <= nums[i] <= 10^9`
