# Result for Longest Substring with At Most K Distinct Characters
# *Kết quả cho bài toán Chuỗi con Dài nhất với Tối đa K Ký tự Khác nhau*

## Description
## *Mô tả*

Given a string `s` and an integer `k`, return *the length of the longest substring of* `s` *that contains at most* `k` *distinct characters*.
*Cho một chuỗi `s` và một số nguyên `k`, trả về *độ dài của chuỗi con dài nhất của* `s` *chứa tối đa* `k` *ký tự khác nhau*.*

## Example 1:
## *Ví dụ 1:*

**Input:** `s = "eceba", k = 2`
**Output:** `3`
**Explanation:** The substring is "ece" with length 3.

## Example 2:
## *Ví dụ 2:*

**Input:** `s = "aa", k = 1`
**Output:** `2`
**Explanation:** The substring is "aa" with length 2.

## Constraints:
## *Ràng buộc:*

*   `1 <= s.length <= 5 * 10^4`
*   `0 <= k <= 50`
