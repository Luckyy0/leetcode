# Result for Prime Palindrome
# *Kết quả cho bài toán Số nguyên tố Đối xứng*

## Description
## *Mô tả*

Given an integer `n`, return *the smallest **prime palindrome** greater than or equal to `n`*.
*Cho một số nguyên `n`, hãy trả về *số nguyên tố đối xứng nhỏ nhất lớn hơn hoặc bằng `n`*.*

An integer is **prime** if it has exactly two divisors: `1` and itself. Note that `1` is not a prime number.
*Một số nguyên là **nguyên tố** nếu nó có đúng hai ước số: `1` và chính nó. Lưu ý rằng `1` không phải là số nguyên tố.*

- For example, `2, 3, 5, 7, 11, and 13` are all primes.

An integer is a **palindrome** if it reads the same from left to right as from right to left.
*Một số nguyên là **đối xứng (palindrome)** nếu nó được đọc từ trái sang phải giống như từ phải sang trái.*

- For example, `101 and 12321` are palindromes.

The test cases are generated so that the answer always exists and is less than or equal to `2 * 10^8`.
*Các trường hợp kiểm thử được tạo ra sao cho câu trả lời luôn tồn tại và nhỏ hơn hoặc bằng `2 * 10^8`.*

---

## Example 1:
## *Ví dụ 1:*

**Input:** n = 6
**Output:** 7

## Example 2:
## *Ví dụ 2:*

**Input:** n = 8
**Output:** 11

## Example 3:
## *Ví dụ 3:*

**Input:** n = 13
**Output:** 101

---

## Constraints:
## *Ràng buộc:*

*   `1 <= n <= 10^8`
